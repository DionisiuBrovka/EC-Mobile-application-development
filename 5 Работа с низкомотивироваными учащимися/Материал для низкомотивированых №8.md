# Работа с базой данных SQLite

## Цель и задачи занятия
- Понять, что такое SQLite и где его используют.
- Научиться создавать таблицы, добавлять, читать и изменять данные.
- Попробовать написать простую программу, которая хранит и выводит информацию.

## Почему это важно?
База данных — это как библиотека, но для компьютера. В реальной работе программисты часто хранят данные о клиентах, заказах, оценках и т.п. Умение быстро создавать и управлять базой позволяет писать более надёжные и масштабируемые программы.

## Основные понятия и идеи
1. **SQLite** – встроенная СУБД, не требует отдельного сервера.
2. **База данных (database)** – файл, в котором хранятся таблицы.
3. **Таблица (table)** – структура, как лист таблицы Excel.
4. **Запись (row)** – одна строка таблицы.
5. **Столбец (column)** – поле с типом данных, например `INTEGER`, `TEXT`.
6. **SQL** – язык запросов, позволяющий читать и менять данные.
7. **Курсор (cursor)** – объект, через который выполняются запросы.

## Объяснение темы

### [10 мин] Введение в SQLite
- SQLite – файл `.db`, открывается через `sqlite3` в Python.
- Пример создания соединения:
  ```python
  import sqlite3
  conn = sqlite3.connect('example.db')
  cur = conn.cursor()
  ```
- Почему это удобно: один файл, быстро запускается, не требует установки сервера.

### [15 мин] Создание таблицы
- Таблица – как таблица в Excel: колонки и строки.
- Пример создания таблицы `students`:
  ```sql
  CREATE TABLE students (
      id INTEGER PRIMARY KEY,
      name TEXT NOT NULL,
      age INTEGER,
      grade TEXT
  );
  ```
- Пояснение: `PRIMARY KEY` гарантирует уникальность `id`.

### [15 мин] Добавление данных
- Вставка одной записи:
  ```sql
  INSERT INTO students (name, age, grade) VALUES ('Alice', 20, 'A');
  ```
- Множественная вставка:
  ```sql
  INSERT INTO students (name, age, grade) VALUES
      ('Bob', 22, 'B'),
      ('Carol', 19, 'A-');
  ```

### [10 мин] Чтение данных
- Выбрать все записи:
  ```sql
  SELECT * FROM students;
  ```
- Выбрать конкретную строку:
  ```sql
  SELECT * FROM students WHERE name = 'Alice';
  ```
- Показать только нужные столбцы:
  ```sql
  SELECT name, grade FROM students;
  ```

### [10 мин] Обновление и удаление
- Обновить запись:
  ```sql
  UPDATE students SET age = 21 WHERE name = 'Alice';
  ```
- Удалить запись:
  ```sql
  DELETE FROM students WHERE name = 'Bob';
  ```

### [10 мин] Работа с Python
- Интеграция всех запросов в один скрипт:
  ```python
  import sqlite3

  conn = sqlite3.connect('students.db')
  cur = conn.cursor()

  cur.execute('''
      CREATE TABLE IF NOT EXISTS students (
          id INTEGER PRIMARY KEY,
          name TEXT NOT NULL,
          age INTEGER,
          grade TEXT
      );
  ''')

  cur.execute("INSERT INTO students (name, age, grade) VALUES ('Dora', 23, 'B+');")
  conn.commit()

  cur.execute("SELECT * FROM students;")
  rows = cur.fetchall()
  for row in rows:
      print(row)

  conn.close()
  ```
