# Управляющие инструкции

## Цель и задачи занятия
- Понять, как Python решает, кто делает что и когда.  
- Научиться пользоваться `if`, `for`, `while` и `break`.  
- Убедиться, что можно управлять потоком программы без «потерянных» строк.  
- Получить навыки писать простые скрипты, которые реагируют на ввод пользователя.

## Почему это важно?
В реальной работе программирование — это не набор команд, а логика.  
Управляющие инструкции позволяют:  
- **Сделать программу умнее** – реагировать на разные ситуации.  
- **Сократить код** – одно условие вместо десятка копий.  
- **Улучшить читаемость** – легче понять, что делает ваш скрипт.

## Основные понятия и идеи
| Термин | Что это? |
|--------|----------|
| **Условие** (`if`) | «Если… тогда…» – выбирает ветку выполнения. |
| **Цикл** (`for`) | Повторяет блок кода несколько раз, «потягивая» элементы из списка. |
| **Цикл** (`while`) | Повторяет, пока условие истинно, как «повторяй, пока не закончится вода». |
| **Выход из цикла** (`break`) | Прерывает цикл раньше времени, как «выход из очереди». |
| **Продолжить** (`continue`) | Пропускает оставшуюся часть итерации, как «пропустить этот шаг». |
| **Блок кода** | Группа инструкций, ограниченная отступом. |
| **Отступ** | В Python ключевой для обозначения блока. |

## Объяснение темы

### 1. Условные операторы [15 мин]
- **Синтаксис**  
  ```python
  if условие:
      действие1
  else:
      действие2
  ```
- **Пример** – проверка возраста для покупки билета.  
  ```python
  age = int(input("Сколько вам лет? "))
  if age >= 18:
      print("Вход разрешён.")
  else:
      print("Пожалуйста, покажите документы.")
  ```
- **Mini‑задание**  
  > Попробуй сам!  
  > Что выведет программа, если пользователь введёт `17`?  
  > Что если `18`?  

### 2. Циклы `for` [20 мин]
- **Синтаксис**  
  ```python
  for элемент in последовательность:
      действие
  ```
- **Пример** – печать всех чисел от 1 до 5.  
  ```python
  for i in range(1, 6):
      print(i)
  ```
- **Аналогия** – «Список покупок»: берём каждый товар, проверяем наличие.  
- **Mini‑задание**  
  > Попробуй сам!  
  > Как изменить код, чтобы вывести только чётные числа?  

### 3. Циклы `while` [15 мин]
- **Синтаксис**  
  ```python
  while условие:
      действие
  ```
- **Пример** – ожидание ввода правильного пароля.  
  ```python
  attempts = 0
  while attempts < 3:
      pwd = input("Пароль: ")
      if pwd == "secret":
          print("Доступ разрешён")
          break
      attempts += 1
  else:
      print("Блокировка")
  ```
- **Mini‑задание**  
  > Попробуй сам!  
  > Что произойдёт, если убрать `break`?  

### 4. Управление циклом: `break` и `continue` [10 мин]
- **`break`** – выходит из ближайшего цикла.  
  ```python
  for num in range(10):
      if num == 5:
          break
      print(num)
  ```
- **`continue`** – пропускает оставшуюся часть текущей итерации.  
  ```python
  for num in range(5):
      if num % 2 == 0:
          continue
      print(num)
  ```
- **Mini‑задание**  
  > Попробуй сам!  
  > Как изменить код, чтобы вывести только числа, не делящиеся на 3?
