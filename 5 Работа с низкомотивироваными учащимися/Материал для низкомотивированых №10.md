# Система ввода‑вывода. Подготовка к ОКР №1

## Цель и задачи занятия  
- Понять, как Python читает данные с клавиатуры и выводит их на экран.  
- Научиться использовать функции `input()`, `print()`, `open()` и контекстные менеджеры.  
- Практиковаться в работе с файлами: чтение, запись, дозапись.  
- Подготовиться к проверке знаний по теме ОКР №1.

## Почему это важно?  
В реальной работе программист всегда взаимодействует с пользователями и внешними источниками: ввод с клавиатуры, чтение конфигурационных файлов, логирование. Умение быстро и надёжно работать с вводом‑выводом повышает скорость разработки и качество кода.

## Основные понятия и идеи  
| Термин | Что это значит? |
|--------|----------------|
| **Ввод** (`input`) | Получаем данные от пользователя в реальном времени. |
| **Вывод** (`print`) | Отображаем информацию на экране. |
| **Файл** | Хранится на диске; можно читать и писать. |
| **Режим чтения** (`'r'`) | Только чтение. |
| **Режим записи** (`'w'`) | Создаём новый файл или перезаписываем. |
| **Режим дозаписи** (`'a'`) | Добавляем данные в конец файла. |
| **Контекстный менеджер** (`with`) | Автоматически закрывает файл, даже если возникла ошибка. |

## Объяснение темы

### 1. Ввод и вывод на экран  [10 мин]  
- **`print()`**: просто печатает строку.  
- **`input()`**: ждёт, пока пользователь введёт текст, и возвращает его как строку.  

```python
name = input("Введите ваше имя: ")
print(f"Привет, {name}!")
```

### 2. Работа с файлами  [25 мин]  
- Открываем файл: `open('data.txt', 'r')`.  
- Читаем: `read()`, `readline()`, `readlines()`.  
- Записываем: `write()`.  
- Дозаписываем: `open('data.txt', 'a')`.  

```python
# Запись
with open('log.txt', 'w') as f:
    f.write('Начало программы\n')

# Добавление
with open('log.txt', 'a') as f:
    f.write('Выполнено действие\n')
```

### 3. Форматирование вывода  [15 мин]  
- Используем f‑строки, `%` и `format()`.  
- Округление, выравнивание, запятая как разделитель.  

```python
pi = 3.1415926535
print(f"Pi ≈ {pi:.3f}")  # 3.142
```

### 4. Ошибки ввода и работа с исключениями  [15 мин]  
- Проверяем ввод: `try/except`.  
- Обрабатываем `ValueError` при преобразовании строки в число.  

```python
try:
    age = int(input("Сколько вам лет? "))
except ValueError:
    print("Пожалуйста, введите целое число.")
```
