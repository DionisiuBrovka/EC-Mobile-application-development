# Реализация основных и дополнительных функциональностей

## Цель и задачи занятия
- Понять, как в Python реализовать базовые и расширенные возможности программы.  
- Научиться использовать функции, модули и классы для более гибкого кода.  
- Узнать, как подключать внешние библиотеки и работать с файлами.  
- Развить уверенность в том, что можно быстро добавлять новые функции в проект.

## Почему это важно?
В реальной работе программист не пишет одну‑разовую программу.  
Нужно быстро дополнять проект новыми возможностями, исправлять ошибки и интегрировать сторонние сервисы.  
Знание, как правильно структурировать код и подключать внешние библиотеки, экономит время и делает проект надёжным.

## Основные понятия и идеи
1. **Функция** – небольшая часть кода, которую можно вызывать многократно.  
2. **Модуль** – файл с набором функций и переменных, который можно импортировать в другие файлы.  
3. **Класс** – шаблон, из которого создаются объекты со своими свойствами и методами.  
4. **Библиотека** – готовый набор модулей, созданных сторонними разработчиками.  
5. **Пакет** – папка с модулями, которая может содержать свой `__init__.py`.  
6. **Условные операторы** – `if`, `else`, `elif` позволяют принимать решения в коде.  
7. **Исключения** – способ обрабатывать ошибки, чтобы программа не падала.

## Объяснение темы

### 1. [10 мин] Функции и аргументы  
- Что такое функция?  
- Как объявлять и вызывать функцию.  
- Позиционные и именованные аргументы.  

```python
def greet(name, greeting="Привет"):
    return f"{greeting}, {name}!"

print(greet("Алиса"))          # Привет, Алиса!
print(greet("Борис", "Здравствуй"))  # Здравствуй, Борис!
```

### 2. [15 мин] Модули и импорт  
- Как создать собственный модуль (`utils.py`).  
- Как импортировать функции и переменные.  

```python
# utils.py
PI = 3.14159

def circle_area(r):
    return PI * r * r
```

```python
# main.py
from utils import PI, circle_area

print(circle_area(5))  # 78.53975
```

### 3. [15 мин] Классы и объекты  
- Определяем класс `Car`.  
- Конструктор `__init__`, методы и атрибуты.  

```python
class Car:
    def __init__(self, make, year):
        self.make = make
        self.year = year

    def age(self, current_year):
        return current_year - self.year

my_car = Car("Toyota", 2015)
print(my_car.age(2024))  # 9
```

### 4. [10 мин] Работа с файлами  
- Чтение из текстового файла.  
- Запись данных в файл.

with open("data.txt", "w") as f:

with open("data.txt", "r") as f:

### 6. [10 мин] Исключения и отладка  
- Что такое `try/except`.  
- Как выводить сообщение об ошибке.  

```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Ошибка: {e}")  # Ошибка: division by zero
```
