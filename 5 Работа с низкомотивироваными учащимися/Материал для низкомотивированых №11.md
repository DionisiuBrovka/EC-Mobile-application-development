# Виджеты и жесты

## Цель и задачи занятия  
- Понять, что такое виджеты и жесты в интерфейсах.  
- Научиться создавать простые виджеты на Python.  
- Освоить основные типы жестов и их обработку.  
- Применить знания в небольших практических задачах.

## Почему это важно?  
Виджеты и жесты делают программы понятными и удобными.  
В реальной жизни они помогают быстро управлять приложением:  
- Нажатие кнопки → открытие меню.  
- Перетаскивание → перемещение элементов.  
- Жесты пальцев → управление смартфоном.  
Знание этих механизмов открывает дорогу к созданию интерактивных игр, приложений и веб‑сайтов.

## Основные понятия и идеи  
1. **Виджет** – элемент интерфейса (кнопка, поле ввода, слайдер).  
2. **Код виджета** – реализация виджета в языке (Python, Tkinter, PyQt).  
3. **Событие** – действие пользователя (клик, ввод текста, перемещение мыши).  
4. **Обработчик события** – функция, которая реагирует на событие.  
5. **Жест** – комбинация движений (пинч, свайп, двойной клик).  
6. **Конвейер событий** – цепочка, в которой события проходят от окна к виджетам.  
7. **Библиотека GUI** – набор готовых виджетов (Tkinter, PyQt, Kivy).

## Объяснение темы  

### [15 мин] Что такое виджеты?  
- Виджет – это как игрушка в наборе.  
- В Python часто используют Tkinter: `Button`, `Label`, `Entry`.  

```python
import tkinter as tk

root = tk.Tk()
btn = tk.Button(root, text="Нажми меня")
btn.pack()
root.mainloop()
```

### [20 мин] Как реагировать на события  
- Событие: клик мыши на кнопке.  
- Обработчик: функция, привязанная к событию.  

```python
def on_click():
    print("Кнопка нажата")

btn.bind("<Button-1>", lambda e: on_click())
```

### [20 мин] Жесты и их применение  
- Жесты: двойной клик, удержание, перетаскивание.  
- В Tkinter: `<Double-Button-1>`, `<B1-Motion>`.

```python
def on_drag(event):
    print(f"Перетаскивание: {event.x}, {event.y}")

root.bind("<B1-Motion>", on_drag)
```

### [15 мин] Практическая мини‑игра  
- Сделаем «Кликер»: нажимай кнопку, счёт увеличивается.  
- Используем `IntVar` для счёта.  

```python
score = tk.IntVar(value=0)

def increment():
    score.set(score.get() + 1)

btn = tk.Button(root, text="Нажми", command=increment)
label = tk.Label(root, textvariable=score)
btn.pack()
label.pack()
```
