# Классы и объекты

## Цель и задачи занятия
- Понять, что такое классы и объекты в Python.  
- Научиться объявлять классы, создавать объекты и работать с их атрибутами и методами.  
- Убедиться, что ООП помогает упорядочить код и облегчает повторное использование.  
- Выполнить несколько практических заданий, чтобы закрепить материал.

## Почему это важно?
В реальной работе программисты часто сталкиваются с большими проектами, где нужно держать код читабельным и модульным.  
Классы позволяют описать «шаблон» для объектов (например, «Автомобиль»), а сами объекты представляют конкретные экземпляры (Toyota, Honda).  
Это делает код легче поддерживать, расширять и тестировать.

## Основные понятия и идеи
- **Класс** – шаблон, описывающий свойства и поведение объектов.  
- **Объект** – конкретная копия класса, с собственными данными.  
- **Атрибут** – переменная, принадлежащая объекту (например, `color`).  
- **Метод** – функция внутри класса, работающая с объектом (например, `drive`).  
- **Конструктор (`__init__`)** – специальный метод, вызываемый при создании объекта.  
- **Наследование** – механизм, позволяющий создать новый класс, основанный на существующем.  
- **Инкапсуляция** – скрытие внутренней реализации объекта, доступ через методы.

## Объяснение темы

### 1. Создание простого класса (≈10 мин)
```python
class Car:
    def __init__(self, brand, color):
        self.brand = brand   # атрибут
        self.color = color   # атрибут

    def drive(self):
        print(f"{self.brand} ({self.color}) едет по дороге.")
```
- `__init__` заполняет атрибуты при создании.  
- `self` указывает на конкретный объект.

### 2. Создание объектов и работа с атрибутами (≈10 мин)
```python
my_car = Car("Toyota", "red")
my_car.drive()          # Выводит: Toyota (red) едет по дороге.
print(my_car.color)     # Выводит: red
```
- Каждый объект хранит свои данные, независимы от других.

### 3. Методы и логика (≈15 мин)
```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.balance = balance

    def deposit(self, amount):
        self.balance += amount
        print(f"{amount} добавлено. Баланс: {self.balance}")

    def withdraw(self, amount):
        if amount <= self.balance:
            self.balance -= amount
            print(f"{amount} снято. Баланс: {self.balance}")
        else:
            print("Недостаточно средств.")
```
- Методы инкапсулируют операции, которые можно выполнять с объектом.

### 4. Наследование и переопределение (≈10 мин)
```python
class SavingsAccount(BankAccount):
    def __init__(self, owner, balance=0, interest_rate=0.01):
        super().__init__(owner, balance)
        self.interest_rate = interest_rate

    def apply_interest(self):
        interest = self.balance * self.interest_rate
        self.deposit(interest)
```
- `SavingsAccount` наследует все методы `BankAccount`, но добавляет новый.
