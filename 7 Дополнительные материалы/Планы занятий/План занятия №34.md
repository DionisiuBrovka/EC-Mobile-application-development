# Лабораторная работа № 19: Создание приложения для Android

## Цель занятия  
1. Научить студентов разрабатывать базовое Android‑приложение, используя Android Studio и язык Kotlin.  
2. Развить навыки работы с интерфейсом пользователя, обработкой событий и сохранением данных.

## Задачи занятия  
- Ознакомиться с основами Android SDK и Android Studio.  
- Создать простое приложение «Калькулятор» с пользовательским интерфейсом.  
- Реализовать обработку событий кнопок и вывод результата.  
- Сохранить вычисленные значения в `SharedPreferences`.  
- Провести отладку и тестирование готового приложения.

## План занятия (90 мин)

- **Организационный момент** — 5 мин  
- **Актуализация знаний** — 10 мин  
- **Изучение нового материала** — 25 мин  
- **Практическая работа** — 30 мин  
- **Закрепление** — 10 мин  
- **Итог занятия** — 10 мин  

## Содержание этапов  

- **Организационный момент**  
  Преподаватель приветствует студентов, проверяет наличие ноутбуков и установку Android Studio, уточняет цели занятия.

- **Актуализация знаний**  
  Быстрый опрос по темам: Activity, View, Intent, XML‑разметка. Студенты отвечают на вопросы, чтобы активировать предшествующие знания.

- **Изучение нового материала**  
  Пояснение структуры проекта в Android Studio, обзор компонентов `Activity`, `Layout`, `ViewModel`, а также объяснение роли `SharedPreferences`. Демонстрация создания нового проекта.

- **Практическая работа**  
  Студенты самостоятельно создают приложение «Калькулятор»:  
  1. Создают макет XML с кнопками и TextView.  
  2. Реализуют обработчики событий кнопок в Kotlin.  
  3. Добавляют сохранение результата в `SharedPreferences`.  
  4. Тестируют работу приложения на эмуляторе.

- **Закрепление**  
  Короткий тест (5 вопросов) по ключевым понятиям и практическая проверка кода: преподаватель просит студентов показать экран кода и объяснить работу.

- **Итог занятия**  
  Обсуждение проблем, которые возникли, подведение итогов, раздача чек‑листов с ключевыми шагами разработки.

## Оборудование и материалы  

- 2 ПК со стабильным интернетом (желательно с Android Studio 4.x+).  
- Проектор и экран для демонстрации.  
- Копии шпаргалок по Android SDK и Kotlin.  
- Учебник «Инструментальное программное обеспечение (изучение Python)» (раздел о работе с IDE).  
- Ссылка на официальный гайд Android: https://developer.android.com/guide.

## Контроль и оценивание  

- **Формативный контроль**: ответы на вопросы в ходе актуализации, проверка кода в процессе практики.  
- **Суммативный контроль**: 5‑вопросный тест по итогам занятия, оценка готового приложения по критериям: корректность UI, работа логики, сохранение данных.  
- Оценка по шкале: 0–10 баллов.

## Домашнее задание  

1. Расширить приложение «Калькулятор» добавлением функционала памяти (сохранить/вызвать число).  
2. Подготовить краткое отчётное задание (1 стр. A4) о процессе разработки, перечислив основные шаги и возникшие трудности.
