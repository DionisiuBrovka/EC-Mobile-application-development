# Лабораторная работа № 21 «Обработка жизненного цикла активности»

## Цель занятия
1. Научить студентов использовать контекстные менеджеры и методы `__enter__`, `__exit__` для управления ресурсами.
2. Познакомить с концепцией жизненного цикла активности в Python‑приложениях и научить применять её на практике.

## Задачи занятия
- Понять, что такое жизненный цикл активности и зачем он нужен.
- Изучить синтаксис и семантику контекстных менеджеров.
- Реализовать собственный контекстный менеджер для управления ресурсами.
- Оценить эффективность использования контекстных менеджеров в реальных сценариях.

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 20 мин  
- Практическое задание (часть 1) — 20 мин  
- Практическое задание (часть 2) — 20 мин  
- Итог занятия — 15 мин  

## Содержание этапов

### Организационный момент
Краткое приветствие, проверка готовности оборудования, объявление темы и целей занятия.

### Актуализация знаний
Повторение принципов работы с ресурсами в Python, знакомство с ранее изученными конструкциями `with` и `try/except`.

### Изучение нового материала
Объяснение концепции жизненного цикла активности, детальный разбор методов `__enter__` и `__exit__`, примеры из стандартной библиотеки.

### Практическое задание (часть 1)
Студенты реализуют простой контекстный менеджер для работы с файловой системой, проверяя корректность открытия и закрытия файлов.

### Практическое задание (часть 2)
Расширение предыдущего примера: добавление логирования, измерения времени выполнения и обработки исключений внутри контекстного менеджера.

### Итог занятия
Обсуждение результатов, ответы на вопросы, подведение итогов, формулирование выводов по применению жизненного цикла активности.

## Оборудование и материалы
- ПК с установленным Python 3.10+ и IDE (PyCharm, VS Code или Jupyter Notebook).
- Проектор/экран для демонстрации кода.
- Учебник «Инструментальное программное обеспечение (изучение Python)» – главы 12‑13.
- Лабораторный скрипт‑шаблон для практических заданий.

## Контроль и оценивание
- Вопросы устно во время практики (проверка понимания концепции).
- Краткий тест (5 вопросов) в конце занятия по работе с контекстными менеджерами.
- Оценка выполнения практического задания (корректность кода, логика работы).

## Домашнее задание
1. Написать контекстный менеджер для работы с сетью (подключение, отправка данных, закрытие соединения).
2. Составить краткое описание (2‑3 строки) о том, где в реальных проектах можно применить жизненный цикл активности.
