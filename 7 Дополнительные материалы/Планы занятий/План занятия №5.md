# Управляющие инструкции

## Цель занятия  
- Освоить основные управляющие конструкции языка Python (условные операторы, циклы).  
- Научиться применять эти конструкции для решения типовых задач.

## Задачи занятия  
1. Понять синтаксис и семантику условных операторов `if`, `elif`, `else`.  
2. Изучить циклы `for` и `while`, условия выхода и вложенность.  
3. Развить навыки комбинирования управляющих конструкций в небольших скриптах.  
4. Применить полученные знания на практических примерах.  
5. Оценить собственные навыки через контрольные вопросы и мини‑проект.

## План занятия  
- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала: условные операторы — 20 мин  
- Изучение нового материала: циклы — 20 мин  
- Практическое закрепление — 15 мин  
- Итог занятия — 5 мин  

## Содержание этапов  
- **Организационный момент**  
  Приветствие, проверка присутствия, короткая беседа о целях урока.  

- **Актуализация знаний**  
  Быстрый опрос по предыдущим темам: функции, списки, словари.  

- **Изучение нового материала: условные операторы**  
  Рассмотрение синтаксиса `if`, `elif`, `else`, логических условий, примеры с вводом пользователя.  

- **Изучение нового материала: циклы**  
  Объяснение `for` и `while`, диапазон `range`, вложенные циклы, `break` и `continue`.  

- **Практическое закрепление**  
  Упражнение: написать программу, выводящую таблицу умножения для заданного числа.  

- **Итог занятия**  
  Подведение итогов, ответы на вопросы, разъяснение домашнего задания.  

## Оборудование и материалы  
- Компьютеры с установленным Python 3.x и IDE (PyCharm, VS Code или IDLE).  
- Проектор/дисплей для демонстрации кода.  
- Рабочая тетрадь/лист с задачами.  
- Учебник/слайды по теме.  

## Контроль и оценивание  
- Вопросы устного опроса в ходе занятия (2–3 вопроса).  
- Мини‑тест из 5 вопросов после изучения циклов.  
- Проверка выполнения практического задания (корректность вывода, отсутствие ошибок).  

## Домашнее задание  
1. Написать программу, которая запрашивает у пользователя число и выводит список всех простых чисел до него.  
2. Реализовать простую игру «угадай число» с использованием цикла `while` и условных операторов.
