# Лабораторная работа № 17: Разработка программ создания файлов

## Цель занятия
1. Научить студентов проектировать и реализовывать Python‑программы, создающие текстовые и бинарные файлы с учётом структуры данных.
2. Развить навыки работы с модулем `os`, `json`, `csv` и `pickle` для управления файловой системой и сериализации данных.

## Задачи занятия
- Ознакомиться с требованиями к формату создаваемых файлов.
- Практически освоить функции работы с файлами: открытие, запись, закрытие, проверка существования.
- Реализовать программу, сохраняющую данные в текстовый, CSV, JSON и бинарный файл.
- Проверить корректность сохранения и чтения файлов.

## План занятия (90 мин)

- **Организационный момент** — 5 мин  
- **Актуализация знаний** — 10 мин  
- **Изучение нового материала** — 25 мин  
- **Практическая работа** — 35 мин  
- **Закрепление** — 10 мин  
- **Итог занятия** — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Преподаватель приветствует группу, проверяет наличие ноутбуков и установленных IDE. Распределяется рабочий лист с описанием задачи.

- **Актуализация знаний**  
  Быстрый опрос студентов о принципах работы с файлами в Python: методы `open`, `write`, `read`, контекстные менеджеры. Уточнение разницы между текстовыми и бинарными файлами.

- **Изучение нового материала**  
  Преподаватель вводит модули `os`, `json`, `csv`, `pickle`. Поясняется структура данных, которые будут сохраняться, и формат файлов. Демонстрация примеров записи и чтения каждого типа.

- **Практическая работа**  
  Студенты в парах пишут программу, которая:
  1. Создаёт директорию `output` (если не существует).
  2. Сохраняет список словарей в текстовый файл (`.txt`), CSV (`.csv`), JSON (`.json`) и бинарный файл (`.pkl`).
  3. Считывает данные обратно и выводит их на экран.
  Преподаватель巡回 по классу, отвечает на вопросы и помогает с отладкой.

- **Закрепление**  
  Быстрый контрольный тест из 5 вопросов: выбор правильного метода сохранения, определение формата файла по расширению, разница между `write` и `writelines`, корректное закрытие файла, назначение модуля `pickle`.

- **Итог занятия**  
  Обсуждение результатов практики, разбор типичных ошибок. Подведение итогов, объявление домашнего задания.

## Оборудование и материалы
- Лэптопы/ПК с установленной IDE (PyCharm, VS Code или Jupyter Notebook).
- Проектор и экран для демонстраций.
- Учебник «Инструментальное программное обеспечение (изучение Python)» — глава о работе с файлами.
- Рабочие листы с задачей и шаблонами кода.

## Контроль и оценивание
- Формативный контроль: ответы на вопросы в ходе занятия.
- Диагностический тест (5 вопросов) после закрепления.
- Оценка практической работы по критериям: корректность кода, соблюдение структуры, работа с разными форматами, читаемость кода.

## Домашнее задание
1. Реализовать программу, которая читает данные из CSV-файла, преобразует их в список словарей и сохраняет в JSON‑файл.
2. Написать скрипт, который проверяет наличие файла `config.ini` в домашней директории, создаёт его со значениями по умолчанию, если он отсутствует.
