# Введение. Объектно‑ориентированное программирование и язык программирования Java

## Цель занятия
- Познакомить студентов с принципами объектно‑ориентированного программирования (ООП) и историей развития языка Java.
- Показать основные синтаксические конструкции Java и их применение в практических задачах.

## Задачи занятия
1. Рассмотреть фундаментальные концепции ООП: инкапсуляция, наследование, полиморфизм и абстракцию.
2. Ознакомить с историей Java и его ролью в современном программировании.
3. Изучить базовые синтаксические элементы языка Java (классы, методы, переменные, условные конструкции).
4. Показать примеры простых программ и их структуру.
5. Оценить уровень усвоения материала через интерактивные вопросы и практическое задание.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- История Java и основные концепции ООП — 20 мин  
- Синтаксис языка Java: классы и методы — 20 мин  
- Пример программы и практическое задание — 20 мин  
- Итог занятия и ответы на вопросы — 5 мин  

## Содержание этапов

**Организационный момент**  
Краткое приветствие, проверка присутствия, объявление темы и целей занятия, раздача листов с планом.

**Актуализация знаний**  
Вопросы по предыдущему материалу о Python, сравнение процедурного и объектно‑ориентированного подходов, обсуждение преимуществ ООП.

**История Java и основные концепции ООП**  
Краткая лекция о создании Java, его эволюции, ключевых особенностях языка и роли ООП в современном ПО. Рассмотрение инкапсуляции, наследования, полиморфизма и абстракции на примерах.

**Синтаксис языка Java: классы и методы**  
Обзор структуры программы на Java, объявление класса, метода `main`, переменных, условных операторов. Демонстрация компиляции и запуска в IDE.

**Пример программы и практическое задание**  
Показ простого калькулятора на Java, разбор кода. Студенты пишут программу «Hello, World!» и простую функцию, используя изученные конструкции.

**Итог занятия и ответы на вопросы**  
Повтор ключевых понятий, ответы на вопросы, уточнение домашнего задания.

## Оборудование и материалы
- Компьютерные классы с установленной IDE (IntelliJ IDEA, Eclipse, NetBeans) и JDK.
- Проектор и экран для демонстрации кода.
- Учебник/практикум по Java (студенты и преподаватель).
- Печатные листы с планом занятия и примерами кода.

## Контроль и оценивание
- Быстрый устный опрос после каждого блока (5–10 вопросов).
- Мини‑тест из 5 вопросов к концу занятия (вариантные ответы).
- Оценка выполнения практического задания (корректность кода, наличие комментариев).

## Домашнее задание
1. Написать программу на Java, которая запрашивает у пользователя два числа и выводит их сумму, разность, произведение и частное.  
2. Составить краткое эссе (200–300 слов) о преимуществах объектно‑ориентированного подхода по сравнению с процедурным, с упоминанием примеров из Python и Java.
