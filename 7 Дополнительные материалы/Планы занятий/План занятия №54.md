# Курсоры. Получение данных и выполнение в фоновом режиме

## Цель занятия
1. Освоить работу с курсорами в Python и научиться использовать их для выборки и обработки данных.  
2. Изучить принципы выполнения операций в фоновом режиме (асинхронность, многопоточность) и применить их к задачам обработки данных.

## Задачи занятия
- Понять роль курсоров в работе с базами данных.  
- Научиться извлекать данные из таблиц с помощью курсоров.  
- Познакомиться с библиотеками `asyncio`, `concurrent.futures` и `threading` для фоновой обработки.  
- Реализовать простую программу, которая одновременно читает данные из БД и выполняет вычисления.  
- Оценить эффективность асинхронного подхода по сравнению с синхронным.

## План занятия (90 мин)

- Организационный момент — 10 мин  
- Актуализация знаний — 15 мин  
- Изучение курсоров — 20 мин  
- Асинхронная обработка данных — 25 мин  
- Практическая работа в группах — 10 мин  
- Итог занятия и обратная связь — 10 мин  

### Содержание этапов

**Организационный момент**  
Краткое приветствие, проверка присутствия, объявление темы и целей урока, распределение студентов по группам.

**Актуализация знаний**  
Повторение основ SQL, работы с базами данных в Python через `sqlite3`, простые запросы SELECT, INSERT.

**Изучение курсоров**  
Пояснение концепции курсора, создание и управление курсорами в `sqlite3`, примеры получения данных построчно, работа с курсорами в больших таблицах.

**Асинхронная обработка данных**  
Введение в `asyncio`, `concurrent.futures.ThreadPoolExecutor`, сравнение с многопоточностью. Демонстрация примера: одновременное чтение из БД и вычисление статистики.

**Практическая работа в группах**  
Студенты реализуют небольшую программу: читают данные из таблицы, обрабатывают их параллельно в потоках, выводят результаты. Педагог проверяет и дает рекомендации.

**Итог занятия и обратная связь**  
Подведение итогов, ответы на вопросы, разбор ошибок, формирование домашнего задания.
