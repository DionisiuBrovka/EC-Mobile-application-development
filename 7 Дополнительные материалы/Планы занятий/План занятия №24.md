# Лабораторная работа № 14  
Обработка исключительных ситуаций

## Цель занятия
1. Познакомить студентов с механизмом обработки исключений в Python.  
2. Научить применять конструкции `try`, `except`, `else` и `finally` в практических задачах.

## Задачи занятия
- Определить типы и источники исключений.  
- Научиться использовать блоки обработки исключений.  
- Реализовать программу, демонстрирующую работу с пользовательскими исключениями.  
- Оценить влияние исключений на логику программы.  
- Разработать собственный обработчик исключений для заданной задачи.

## План занятия
- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 30 мин  
- Практическая часть — 30 мин  
- Закрепление — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов
- **Организационный момент** – приветствие, проверка присутствия, постановка темы.  
- **Актуализация знаний** – краткое повторение предыдущих уроков по условным операторам и циклам, которые необходимы для понимания обработки исключений.  
- **Изучение нового материала** – теоретическое введение в исключения, синтаксис `try/except`, разбор стандартных и пользовательских исключений.  
- **Практическая часть** – выполнение лабораторного задания: написание программы, читающей данные из файла, обрабатывающей возможные ошибки (файл отсутствует, неверный формат) и выводящей итоговый результат.  
- **Закрепление** – групповая работа по решению короткого теста и обсуждение решений.  
- **Итог занятия** – подведение итогов, ответ на вопросы, раздача листовок с примерами кода.  

## Оборудование и материалы
- Компьютерный класс с доступом к интернету.  
- IDE (PyCharm, VS Code или аналог).  
- Проектор и экран для демонстрации кода.  
- Учебный материал: «Python. Полный курс» – раздел об исключениях.  
- Лабораторный пакет с шаблоном кода.  

## Контроль и оценивание
- Форма проверки знаний:  
  - 5 вопросов с выбором ответа (тест) по теории.  
  - Проверка выполненного лабораторного задания (корректность кода, наличие обработки ошибок).  
  - Мини‑дискуссия по ключевым аспектам обработки исключений.  

## Домашнее задание
1. Написать программу, которая запрашивает у пользователя два числа, делит первое на второе и обрабатывает исключение `ZeroDivisionError`.  
2. Создать собственный пользовательский класс исключения `InvalidDataError` и применить его в простом валидаторе входных данных.
