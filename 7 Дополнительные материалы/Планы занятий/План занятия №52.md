# Работа с базой данных SQLite

## Цель занятия
- Научить студентов создавать и управлять базами данных SQLite с использованием Python.
- Развить навыки выполнения CRUD‑операций и написания простых запросов SQL.

## Задачи занятия
1. Ознакомить с концепциями реляционных баз данных и особенностями SQLite.
2. Показать, как подключаться к базе, создавать таблицы и выполнять запросы из Python.
3. Научить анализировать и оптимизировать простые SQL‑операции.
4. Предоставить практическое задание для закрепления навыков.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала (подключение и создание таблиц) — 20 мин  
- Изучение нового материала (CRUD‑операции и запросы) — 20 мин  
- Практическая работа (с заданием) — 20 мин  
- Закрепление и контроль знаний — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Приветствие, проверка присутствия, краткое объявление темы и целей.

- **Актуализация знаний**  
  Быстрый опрос: что такое реляционная база данных, какие типы СУБД существуют, чем отличается SQLite от других СУБД.

- **Изучение нового материала (подключение и создание таблиц)**  
  Демонстрация установки библиотеки `sqlite3`, создания соединения, создания таблицы с помощью SQL‑команды `CREATE TABLE`.

- **Изучение нового материала (CRUD‑операции и запросы)**  
  Пошаговое выполнение операций INSERT, SELECT, UPDATE, DELETE и примеры фильтрации, сортировки и агрегации данных.

- **Практическая работа (с заданием)**  
  Студенты в группах создают простую базу данных «Книги» и реализуют CRUD‑операции через скрипт Python.

- **Закрепление и контроль знаний**  
  Быстрый тест из 5 вопросов + обсуждение ответов, разбор типичных ошибок.

- **Итог занятия**  
  Подведение итогов, напоминание о домашнем задании, ответы на вопросы.

## Оборудование и материалы

- Компьютерный класс с установленным Python 3.10+ и IDE (PyCharm, VS Code или Jupyter Notebook).  
- Проектор/экран для демонстрации кода.  
- Печатные листы с примерами SQL‑команд (необязательно).  
- Учебник/ссылка на официальную документацию `sqlite3` (https://docs.python.org/3/library/sqlite3.html).  

## Контроль и оценивание

- Краткий тест (5 вопросов) в конце занятия (письменный или онлайн).  
- Оценка выполнения практического задания: корректность кода, наличие всех CRUD‑операций, комментарии к коду.  
- Участие в обсуждении и ответы на вопросы.

## Домашнее задание

1. Создать базу данных `students.db` с таблицей `students` (id, name, age, grade).  
2. Написать скрипт, который:
   - Добавляет 5 новых записей.  
   - Выводит всех студентов с оценкой выше 85.  
   - Обновляет возраст одного студента.  
   - Удаляет запись с минимальной оценкой.  

Упражнение должно быть выполнено в виде одного скрипта Python и отправлено в формате `.py`.
