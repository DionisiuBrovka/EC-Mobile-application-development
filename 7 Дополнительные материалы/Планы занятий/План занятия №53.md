# Лабораторная работа № 32: Использование базы данных SQLite

## Цель занятия
- Научить студентов создавать, модифицировать и управлять SQLite‑базами данных при помощи Python.
- Развить навыки работы с SQL‑запросами и библиотекой `sqlite3` в практических задачах.

## Задачи занятия
1. Ознакомиться с принципами работы реляционных баз данных и особенностями SQLite.
2. Научиться создавать таблицы, вставлять, обновлять и удалять данные через SQL‑команды.
3. Освоить подключение к SQLite из Python и выполнение запросов с помощью модуля `sqlite3`.
4. Выполнить практическое задание по проектированию и заполнению базы данных.
5. Оценить эффективность работы с базой данных и выявить типичные ошибки.

## План занятия (90 минут)
- Организационный момент — 5 мин
- Актуализация знаний — 10 мин
- Изучение нового материала — 30 мин
- Практическая работа — 30 мин
- Закрепление и контроль — 10 мин
- Итог занятия — 5 мин

## Содержание этапов
- **Организационный момент**  
  Проверка готовности оборудования, объявление темы, краткое напоминание о предшествующих лабораторных работах.

- **Актуализация знаний**  
  Быстрый опрос студентов о работе с базами данных, обсуждение основных понятий: таблица, строка, первичный ключ, индексы.

- **Изучение нового материала**  
  Лекция по синтаксису SQL для SQLite, демонстрация создания таблиц, индексов, вставки и изменения данных, а также подключение к базе из Python и выполнение запросов через `sqlite3`.

- **Практическая работа**  
  Студенты реализуют проект: создают базу данных `library.db`, таблицы `authors`, `books`, заполняют их данными, выполняют запросы для выборки и обновления записей.

- **Закрепление и контроль**  
  Быстрый тест из 5 вопросов по SQL‑командам и работе с `sqlite3`; обсуждение решений, разбор ошибок.

- **Итог занятия**  
  Подведение итогов, раздача чек‑листов, объявление домашнего задания.

## Оборудование и материалы
- Персональные компьютеры с установленной Python 3.10+ и IDE (PyCharm, VS Code, Jupyter Notebook).
- Проектор и экран для демонстрации кода.
- Студентские ноутбуки с доступом к Интернету для скачивания шаблонов и библиотек.
- Учебник «Python для научных вычислений» (раздел о работе с SQLite).
- Файл‑шаблон `library_template.sql` для ускорения работы.

## Контроль и оценивание
- **Краткий тест** (5 вопросов) после практики: проверка понимания SQL‑операторов и синтаксиса `sqlite3`.
- **Проверка лабораторного задания**: наличие корректной схемы таблиц, выполненные запросы, логика кода.
- **Участие в обсуждении**: активность в процессе лекции и практики.

2. Написать скрипт, который выводит список всех книг с рейтингом выше 4, отсортированный по дате публикации. При сдаче предоставить скрипт и краткое описание используемых запросов.
