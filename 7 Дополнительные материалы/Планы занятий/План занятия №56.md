# Лабораторная работа № 34  
## Реализация курсоров с асинхронными задачами

### Цель занятия  
1. Научить студентов проектировать и реализовывать асинхронные курсоры, работающие с потоками данных.  
2. Развить навыки работы с библиотекой `asyncio` и понимание принципов параллельного выполнения задач в Python.

### Задачи занятия  
1. Ознакомить студентов с концепцией асинхронных итераторов.  
2. Показать практическое применение асинхронных курсоров для чтения больших файлов.  
3. Практиковать отладку и профилирование асинхронного кода.  
4. Разработать собственный асинхронный курсор, интегрированный в учебный проект.  
5. Оценить качество реализации по заданным критериям.

### План занятия (90 мин)  

- [Организационный момент] — 5 мин  
- [Актуализация знаний] — 10 мин  
- [Изучение нового материала] — 25 мин  
- [Практическая работа] — 35 мин  
- [Закрепление и контроль] — 10 мин  
- [Итог занятия] — 5 мин  

#### Содержание этапов  

- **Организационный момент** – проверка присутствия, краткое напоминание о цели и задачах занятия.  
- **Актуализация знаний** – обзор предыдущих лабораторных работ по работе с потоками и синхронными курсорами.  
- **Изучение нового материала** – теоретическое введение в асинхронные итераторы, синтаксис `async for`, основные методы `__aiter__` и `__anext__`.  
- **Практическая работа** – студенты самостоятельно реализуют асинхронный курсор для чтения CSV‑файла, интегрируют его в существующий проект и проверяют корректность работы через набор тестов.  
- **Закрепление и контроль** – краткий тест (3‑4 вопроса) и разбор решений, обсуждение ошибок и альтернативных подходов.  
- **Итог занятия** – подведение итогов, раздача чек‑листов по проверке качества кода, напоминание о домашнем задании.

### Оборудование и материалы  
- Персональные компьютеры со средой разработки (VS Code, PyCharm, Jupyter) и установленным Python 3.10+.  
- Проектор и ноутбук преподавателя для демонстрации кода.  
- Репозиторий GitHub/Bitbucket с шаблоном проекта.  
- Текстовый редактор/IDE и расширения для работы с `asyncio`.  
- Учебные материалы: главы из книги «Python Asynchronous Programming» и статьи по `asyncio`.

### Контроль и оценивание  
- **Тест** (5 мин): 3‑4 вопросов по синтаксису асинхронных итераторов.  
- **Практическое задание**: оценка кода по критериям – корректность, читаемость, покрытие тестами, эффективность.  
- **Обратная связь**: комментарии преподавателя к каждому студенту в системе контроля версий.

### Домашнее задание  
1. Реализовать асинхронный курсор для чтения JSON‑файла с большим объёмом данных и вывести статистику по ключам.  
2. Написать небольшую программу, которая одновременно читает два файла с использованием асинхронных курсоров и выводит суммарную длину строк.
