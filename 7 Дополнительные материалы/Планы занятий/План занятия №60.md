# Создание анимации. Приемники широковещательных намерений

## Цель занятия  
1. Научить студентов основам создания анимации на Python с использованием библиотек для широковещательных намерений.  
2. Познакомить студентов с принципами работы приемников широковещательных сообщений в контексте анимации.

## Задачи занятия  
- Понять теоретические основы анимации и широковещательных намерений.  
- Изучить структуру и синтаксис ключевых библиотек (например, `matplotlib.animation`, `pygame`, `asyncio`).  
- Практически реализовать простую анимацию, реагирующую на события широковещательной сети.  
- Оценить эффективность подходов к обработке широковещательных сообщений в графическом приложении.  

## План занятия  

- Организационный момент — 5 мин  
- Актуализация знаний — 8 мин  
- Теоретическая часть: принципы анимации и широковещательных намерений — 10 мин  
- Демонстрация примера кода — 8 мин  
- Практическое задание: создание анимации с приемником — 10 мин  
- Подведение итогов и ответы на вопросы — 4 мин  

## Содержание этапов  

- **Организационный момент**  
  Введение в тему, проверка присутствия, постановка целей занятия.  

- **Актуализация знаний**  
  Быстрый опрос по предыдущим лекциям о графике в Python, библиотеке `matplotlib`, `pygame`.  

- **Теоретическая часть: принципы анимации и широковещательных намерений**  
  Объяснение циклов анимации, таймеров, событийных циклов, а также концепции широковещательных сообщений и их применение в графических приложениях.  

- **Демонстрация примера кода**  
  Показ готового скрипта, где анимация реагирует на широковещательный сигнал, с разбором ключевых строк.  

- **Практическое задание: создание анимации с приемником**  
  Студенты пишут код, создавая простую анимацию (например, шарик, движущийся по экрану), которая меняет направление при получении широковещательного события.  

- **Подведение итогов и ответы на вопросы**  
  Краткое резюме, уточнение непонятных моментов, оценка понимания студентов.  

## Оборудование и материалы  

- Лэптопы с установленным Python 3.10+, IDE (PyCharm/VSCode).  
- Проектор и экран для демонстрации кода.  
- Доступ к интернету для скачивания библиотек (`pip install matplotlib pygame asyncio`).  

## Контроль и оценивание  

- Краткий письменный тест (5 вопросов) по теории анимации и широковещательных намерений.  
- Проверка кода практического задания: корректность работы анимации, наличие обработки широковещательного сигнала.  

## Домашнее задание  

- Написать программу, где два объекта анимируются одновременно, а их взаимодействие регулируется широковещательным сообщением (например, при столкновении один из объектов меняет цвет).  
- Подготовить краткую заметку (1–2 страницы) о применении широковещательных намерений в других сферах программирования (графика, игры, IoT).
