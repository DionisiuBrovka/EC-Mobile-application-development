# Лабораторная работа № 5: Разработка классов и использование их в программах

## Цель занятия  
1. Освоить принципы объектно‑ориентированного программирования: определение классов, атрибутов, методов и наследования.  
2. Научиться применять созданные классы в реальных задачах, интегрируя их в существующие программы.

## Задачи занятия  
- Ознакомиться с синтаксисом и семантикой объявлений классов в Python.  
- Понять разницу между экземплярами и классами, а также роль `self` и `@classmethod`.  
- Реализовать несколько простых классов, включающих инкапсуляцию и наследование.  
- Интегрировать полученные классы в лабораторный проект, используя их для решения конкретной задачи.  
- Оценить корректность работы программы с помощью тестов и отладки.

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 25 мин  
- Практическая работа — 30 мин  
- Закрепление и контроль — 10 мин  
- Итог занятия — 10 мин  

## Содержание этапов

### Организационный момент  
Краткое приветствие, проверка готовности к работе, раздача инструкций по лабораторной задаче.

### Актуализация знаний  
Обсуждение ключевых понятий из предыдущих занятий: функции, модули, списки, словари. Уточнение вопросов, связанных с синтаксисом.

### Изучение нового материала  
Введение в объектно‑ориентированное программирование: объявление классов, атрибуты экземпляра, методы, `__init__`, `self`, `@staticmethod` и `@classmethod`. Примеры кода, демонстрация наследования и полиморфизма.

### Практическая работа  
Студенты реализуют два класса: `Person` (с атрибутами имя, возраст) и `Student` (наследует `Person`, добавляет номер группы). Затем интегрируют их в проект «Учет студентов», создавая список объектов и выводя информацию.

### Закрепление и контроль  
Краткое тестовое задание: написать метод `age_category` в классе `Person`, возвращающий строку «child», «adult» или «senior» в зависимости от возраста. Проверка кода через автоматические тесты.

### Итог занятия  
Обсуждение результатов работы, ответы на вопросы, подведение итогов, раздача домашнего задания.

## Оборудование и материалы  
- Компьютеры со стабильной установкой Python 3.10+  
- IDE (PyCharm, VS Code, Thonny)  
- Проектор/дисплей для демонстрации кода  
- Слайды/тексты (объектно‑ориентированное программирование)  
- Учебник по Python (статьи/разделы о классах)  

## Контроль и оценивание  
- Проверка выполненного кода на наличие синтаксических ошибок и корректной работы методов.  
- Тесты (pytest) для автоматической проверки функциональности классов.  
- Краткая устная проверка понимания концепций.  

## Домашнее задание  
1. Создать класс `BankAccount` с методами `deposit`, `withdraw`, `balance`.  
2. Написать программу, создающую список аккаунтов, выполняющую операции и выводящую итоговый баланс каждого аккаунта.  
3. Подготовить краткую документацию (docstring) для всех методов.
