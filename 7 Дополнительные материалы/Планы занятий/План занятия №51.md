# Лабораторная работа № 31: Реализация выдвижных панелей

## Цель занятия
1. Научить студентов проектировать и реализовывать графический интерфейс с выдвижными панелями при помощи Python и библиотеки Tkinter.  
2. Развить навыки работы с событиями, виджетами и модулем `ttk` для создания динамических элементов интерфейса.

## Задачи занятия
- Ознакомить с концепцией выдвижных панелей и их применением в GUI.  
- Показать структуру приложения на Tkinter, включая создание `Notebook`, `PanedWindow` и `Toplevel`.  
- Практическая реализация простого интерфейса с несколькими выдвижными панелями.  
- Обсудить способы управления видимостью панелей и сохранения состояния.  
- Проверить понимание студенями работы с событиями и обработчиками.

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Введение в выдвижные панели — 15 мин  
- Демонстрация готового решения — 10 мин  
- Практическая работа студентов — 25 мин  
- Итоги и вопросы — 5 мин  
- Закрепление — 5 мин  

## Содержание этапов

**Организационный момент**  
Проверка присутствия, настройка оборудования, краткое напоминание о правилах лабораторной работы.

**Актуализация знаний**  
Обсуждение предыдущих лабораторных работ, где использовались виджеты `Frame`, `Button`, `Entry`. Быстрый тест на запоминание синтаксиса `pack`, `grid`, `place`.

**Введение в выдвижные панели**  
Объяснение принципов работы выдвижных панелей, разбор примеров из `ttk.Notebook`, `PanedWindow` и `Toplevel`. Показать, как использовать `state="hidden"` и `state="normal"`.

**Демонстрация готового решения**  
Показ готового кода лабораторной работы №31, разбор ключевых моментов: инициализация, добавление панелей, привязка событий, сохранение состояния.

**Практическая работа студентов**  
Студенты самостоятельно реализуют приложение:  
1. Создают `Tk`‑окно.  
2. Добавляют `PanedWindow` с тремя панелями.  
3. Реализуют кнопку «Скрыть/Показать» для каждой панели.  
4. Сохраняют состояние в файл `pickle`.  

**Итоги и вопросы**  
Проверка выполнения задач, ответы на вопросы, разъяснение трудностей.

**Закрепление**  
Короткая проверка: 3 вопроса с выбором ответа о работе с `ttk` и `PanedWindow`.

## Оборудование и материалы
- 10 ПК со стандартным Python 3.10+ и установленной библиотекой Tkinter.  
- Проектор и экран для демонстрации.  
- IDE (PyCharm, VS Code) на каждом компьютере.  
- Лабораторный материал (код-скелет, шаблоны).  
- Копии учебника «Python для начинающих» (глава о Tkinter).  

## Контроль и оценивание
- **Кодовое задание**: проверка на наличие всех заявленных функций и корректной работы интерфейса.  
- **Короткий тест**: 3 вопроса с выбором ответа о свойствах виджетов и событиях.  
- **Устная проверка**: ответы на вопросы по работе с панелями и сохранению состояния.  

## Домашнее задание
1. Расширить приложение, добавив вкладки `Notebook` с выдвижными панелями внутри каждой вкладки.  
2. Реализовать сохранение настроек (положение панелей, открыты/закрыты) в файл JSON и восстановление при следующем запуске.
