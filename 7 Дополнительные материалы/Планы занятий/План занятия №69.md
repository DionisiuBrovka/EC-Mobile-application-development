# Реализация основных и дополнительных функциональностей

## Цель занятия
1. Познакомить студентов с принципами реализации ключевых и вспомогательных функциональностей в Python.  
2. Развить навыки проектирования и отладки модульных компонентов программы.

## Задачи занятия
- Объяснить различия между базовыми и дополнительными функциями.  
- Показать методы тестирования и документирования функций.  
- Практиковать написание функций с использованием параметров по умолчанию и переменного числа аргументов.  
- Научить использовать встроенные и сторонние библиотеки для расширения функциональности.  
- Оценить качество кода по критериям читаемости, эффективности и тестируемости.

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала: основные функции — 20 мин  
- Изучение нового материала: дополнительные функции и расширения — 20 мин  
- Практическая работа: написание функций с тестами — 15 мин  
- Закрепление и обсуждение результатов — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Вступительное слово, проверка присутствия, краткое напоминание о предыдущем занятии.  

- **Актуализация знаний**  
  Быстрый опрос: какие типы функций студенты уже писали, какие проблемы встречались при их использовании.  

- **Изучение нового материала: основные функции**  
  Рассмотрение синтаксиса определения функций, работы с параметрами, возврата значений, области видимости переменных.  

- **Изучение нового материала: дополнительные функции и расширения**  
  Параметры по умолчанию, *args, **kwargs, декораторы, генераторы и итераторы как средства расширения.  

- **Практическая работа: написание функций с тестами**  
  Студенты разбиваются на пары, получают задание: реализовать функцию сортировки, добавить параметры и написать unit‑тесты.  

- **Закрепление и обсуждение результатов**  
  Обзор решений, обсуждение ошибок и лучших практик, ответы на вопросы.  

- **Итог занятия**  
  Краткое резюме, напоминание о домашнем задании, объявление даты следующего урока.  

## Оборудование и материалы

- ПК с установленным Python 3.11+ и IDE (PyCharm, VS Code).  
- Проектор/экран для демонстрации кода.  
- Доступ к репозиторию GitHub с шаблонами тестов.  
- Учебник «Python для начинающих» (дополнительный материал).  

## Контроль и оценивание

- Краткий тест (5 вопросов) в конце занятия по ключевым понятиям.  
- Оценка практической работы по критериям: корректность, читаемость, покрытие тестами.  
- Участие в обсуждении: активность и вопросы.
