# Лабораторная работа № 15 Поточная модель

## Цель и задачи

1. Познакомить студентов с концепцией потоковой (stream‑) модели данных в Python.  
2. Научить реализовывать простейшие потоки чтения, обработки и записи данных.  
3. Развить навыки работы с модулями `io`, `contextlib`, `asyncio` и `itertools`.  
4. Оценить производительность потоковой обработки по сравнению с «загрузить‑в‑память» подходом.

## Теоретическая часть

| Понятие | Определение |
|--------|-------------|
| Поток (stream) | Последовательный доступ к данным, где каждый элемент читается один за другим без необходимости загрузки всех данных в память. |
| Буферизация | Сохранение части данных в временном хранилище (буфере) для уменьшения количества операций ввода‑вывода. |
| Генератор | Функция, возвращающая объект‑итератор, позволяющий лениво вычислять значения. |
| `io.BufferedReader/Writer` | Классы, обеспечивающие буферизацию чтения/записи файлов. |
| `contextlib.closing` | Утилита, автоматически закрывающая объект после выхода из контекста. |
| `asyncio` | Библиотека для асинхронного ввода‑вывода, позволяющая обрабатывать несколько потоков одновременно. |
| `itertools` | Модуль, содержащий эффективные итераторы для работы с последовательностями. |
| Параллельная обработка | Распараллеливание задач для ускорения выполнения при помощи потоков или процессов. |
| `multiprocessing` | Модуль для создания процессов и обмена данными между ними. |
| Производительность | Время выполнения и потребление памяти при работе с большими наборами данных. |

## Ход выполнения работы

1. **Подготовка данных**  
   Создайте текстовый файл `sample.txt` размером ~10 МБ с случайным текстом.  
   ```python
   import random, string
   with open('sample.txt', 'w') as f:
       for _ in range(200000):
           f.write(''.join(random.choices(string.ascii_letters + string.digits, k=50)) + '\n')
   ```

2. **Считывание файла построчно**  
   Реализуйте чтение файла без загрузки в память, используя `io.open` в режиме чтения.  
   ```python
   import io
   with io.open('sample.txt', 'r', encoding='utf-8') as f:
       for line in f:
           process(line)
   ```

3. **Потоковая обработка**  
   Напишите функцию `process(line)` – заменяющую все цифры на `#`.  
   ```python
   import re
   def process(line):
       return re.sub(r'\d', '#', line)
   ```

4. **Запись результата в новый файл**  
   Используйте буферизованный писатель `io.BufferedWriter`.  
   ```python
   with io.open('processed.txt', 'w', encoding='utf-8') as out:
       for line in io.open('sample.txt', 'r', encoding='utf-8'):
           out.write(process(line))
   ```

5. **Сравнение с «загрузить‑в‑память»**  
   *Загрузите весь файл в список*, обработайте его и запишите обратно.  
   ```python
   data = open('sample.txt').readlines()
   processed = [process(l) for l in data]
   open('full_processed.txt', 'w').writelines(processed)
   ```

6. **Анализ производительности**  
   *Измерьте время* (`time.perf_counter`) и потребление памяти (`tracemalloc`) для обоих подходов.  
   ```python
   import time, tracemalloc
   start = time.perf_counter()
   # потоковый вариант
   end = time.perf_counter()
   print(f'Поток: {end-start:.2f}s')
   ```

7. **Асинхронный вариант**  
   Реализуйте чтение/запись через `asyncio` и `aiofiles`.  
   ```python
   import asyncio, aiofiles
   async def async_process():
       async with aiofiles.open('sample.txt', 'r') as f,\
                    aiofiles.open('async_processed.txt', 'w') as out:
           async for line in f:
               await out.write(process(line))
   asyncio.run(async_process())
   ```

8. **Параллельная обработка**  
   Используйте `multiprocessing.Pool` для параллельной обработки блоков данных.  
   ```python
   from multiprocessing import Pool
   def chunk(lines):
       return [process(l) for l in lines]
   with open('sample.txt') as f:
       lines = f.readlines(100000)  # читаем 100 000 строк за раз
   with Pool(4) as pool:
       results = pool.map(chunk, lines)
   ```

9. **Формирование отчёта**  
   Сформируйте таблицу с результатами времени и памяти, добавьте графики.  
   ```python
   import matplotlib.pyplot as plt
   plt.bar(['Поток', 'Полный загруз', 'Async', 'Паррал'], times)
   plt.ylabel('Время, с')
   plt.show()
   ```
