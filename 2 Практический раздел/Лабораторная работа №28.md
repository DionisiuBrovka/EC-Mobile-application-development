# Лабораторная работа № 28  
Создание динамических фрагментов

## Цель и задачи

1. Познакомиться с принципами работы динамических фрагментов в Python.  
2. Научиться создавать и использовать функции, принимающие произвольное число аргументов (*args*, *kwargs*).  
3. Разобрать применение переменных и словарей как динамических структур данных.  
4. Осуществить экспериментальное измерение времени выполнения динамических конструкций.

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| **Функция** | Самостоятельный блок кода, выполняемый по вызову. |
| ***args* | Кортеж позиционных аргументов, передаваемых функции. |
| **kwargs** | Словарь именованных аргументов, передаваемых функции. |
| **Динамический тип** | Тип, который может изменяться во время выполнения программы. |
| **Mutable** | Изменяемый объект (список, словарь). |
| **Immutable** | Неизменяемый объект (целое число, строка). |
| **Lambda** | Анонимная функция, объявляемая в строке. |
| **Generator** | Функция, возвращающая итератор с `yield`. |
| **Decorator** | Функция, принимающая и возвращающая другую функцию. |
| **`timeit`** | Модуль для измерения времени выполнения кода. |

## Ход выполнения работы

1. **Инициализация окружения**  
   *Откройте терминал и создайте новый файл `dynamic_fragments.py`.*  
   ```python
   # dynamic_fragments.py
   import timeit
   ```

2. **Создание функции с `*args`**  
   *Покажите, как функция может принимать произвольное число позиционных аргументов.*  
   ```python
   def sum_all(*args):
       """Возвращает сумму всех переданных чисел."""
       return sum(args)
   ```

3. **Создание функции с `**kwargs`**  
   *Покажите, как функция обрабатывает именованные аргументы.*  
   ```python
   def print_info(**kwargs):
       """Выводит ключ-значение каждой пары."""
       for key, value in kwargs.items():
           print(f'{key}: {value}')
   ```

4. **Комбинирование `*args` и `**kwargs`**  
   *Создайте функцию, принимающую обе категории аргументов.*  
   ```python
   def process_data(*args, **kwargs):
       total = sum_all(*args)
       print_info(**kwargs)
       return total
   ```

5. **Использование динамических словарей**  
   *Покажите, как словарь может быть расширен во время выполнения.*  
   ```python
   config = {}
   config['threshold'] = 0.8
   config.update({'mode': 'auto', 'verbose': True})
   ```

6. **Проверка мутабельности**  
   *Проведите эксперимент с изменением списка внутри функции.*  
   ```python
   def add_item(lst, item):
       lst.append(item)
   my_list = [1, 2, 3]
   add_item(my_list, 4)
   print(my_list)  # [1, 2, 3, 4]
   ```

7. **Измерение времени выполнения**  
   *Сравните время простого цикла с использованием генератора.*  
   ```python
   def generator_sum(n):
       return sum(i for i in range(n))

   def loop_sum(n):
       total = 0
       for i in range(n):
           total += i
       return total

   print(timeit.timeit('generator_sum(1000000)', globals=globals(), number=10))
   print(timeit.timeit('loop_sum(1000000)', globals=globals(), number=10))
   ```

8. **Сохранение результатов**  
   *Запишите выводы в файл `results.txt`.*  
   ```python
   with open('results.txt', 'w') as f:
       f.write(f'Generator time: {timeit.timeit(...)}\n')
       f.write(f'Loop time: {timeit.timeit(...)}\n')
   ```
