# Лабораторная работа № 40  
**Использование виджетов на главном экране**

## Цель и задачи
1. Познакомить студентов с основами создания графического интерфейса в Python.  
2. Научить работать с виджетами (Label, Button, Entry, Checkbutton, RadioButton, Listbox) в библиотеке `tkinter`.  
3. Развить умение комбинировать виджеты в макетах с помощью менеджеров геометрии `pack`, `grid`, `place`.  
4. Провести эксперимент по сбору пользовательского ввода и отображению результатов.

## Теоретическая часть
| Понятие | Определение |
|---------|-------------|
| **GUI** | Графический пользовательский интерфейс – набор графических элементов, через которые пользователь взаимодействует с программой. |
| **Виджет** | Самостоятельный графический элемент интерфейса (кнопка, метка, поле ввода и т.п.). |
| **Tkinter** | Стандартная библиотека Python для создания GUI, основанная на Tcl/Tk. |
| **Менеджер геометрии** | Механизм размещения виджетов внутри контейнера (`pack`, `grid`, `place`). |
| **Callback** | Функция, привязанная к событию (нажатие кнопки, изменение текста и т.п.). |
| **Frame** | Контейнер, который можно использовать для группировки виджетов. |
| **Event loop** | Главный цикл программы, обрабатывающий события пользовательского ввода. |
| **Layout** | Расположение виджетов внутри окна. |
| **Binding** | Привязка события к обработчику. |
| **Widget options** | Параметры виджетов (текст, цвет, размер, стиль). |

## Ход выполнения работы

1. **Подготовка окружения**  
   Откройте любой редактор кода (VS Code, PyCharm, IDLE) и создайте новый файл `lab40.py`.  
   ```python
   import tkinter as tk
   from tkinter import ttk
   ```

2. **Создание главного окна**  
   ```python
   root = tk.Tk()
   root.title("Лабораторная работа № 40")
   root.geometry("400x300")
   ```

3. **Размещение виджетов**  
   - Добавьте `Label` для заголовка.  
   - Создайте `Entry` для ввода имени.  
   - Добавьте `Button`, который будет выводить приветствие в `Label`.  
   ```python
   title = ttk.Label(root, text="Введите ваше имя:", font=("Arial", 12))
   title.pack(pady=10)

   name_var = tk.StringVar()
   entry = ttk.Entry(root, textvariable=name_var)
   entry.pack(pady=5)

   def greet():
       greeting = f"Привет, {name_var.get()}!"
       result_label.config(text=greeting)

   greet_btn = ttk.Button(root, text="Приветствовать", command=greet)
   greet_btn.pack(pady=10)

   result_label = ttk.Label(root, text="", font=("Arial", 10))
   result_label.pack(pady=5)
   ```

4. **Добавление дополнительных виджетов**  
   - `Checkbutton` для выбора опции.  
   - `Radiobutton` для выбора цвета текста.  
   ```python
   option_var = tk.IntVar(value=0)
   check = ttk.Checkbutton(root, text="Показывать сообщение", variable=option_var)
   check.pack(pady=5)

   color_var = tk.StringVar(value="black")
   colors = [("Черный", "black"), ("Красный", "red"), ("Синий", "blue")]
   for text, color in colors:
       rb = ttk.Radiobutton(root, text=text, variable=color_var, value=color)
       rb.pack(anchor='w')
   ```

5. **Модификация обработчика**  
   Учтите состояние чекбокса и выбранный цвет.  
   ```python
   def greet():
       if option_var.get():
           greeting = f"Привет, {name_var.get()}!"
           result_label.config(text=greeting, foreground=color_var.get())
   ```

6. **Запуск Event Loop**  
   ```python
   root.mainloop()
   ```

7. **Экспериментальная часть**  
   - Запустите программу.  
   - Введите имя, отметьте чекбокс, выберите цвет, нажмите кнопку.  
   - Зафиксируйте полученное сообщение и визуальное оформление.  

8. **Анализ результатов**  
   Опишите, как изменение каждого параметра (чекбокс, радиокнопка) влияет на вывод.

7. Опишите разницу между `tkinter` и `ttk` виджетами.  
8. Как добавить в программу обработчик ошибки, если пользователь не вводит имя?  
9. Как сохранить состояние виджетов (например, выбранные радиокнопки) между запуском программы?
