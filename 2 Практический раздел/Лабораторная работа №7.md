# Лабораторная работа № 7  
Перегрузка методов и передача аргументов  

## Цель и задачи  
1. Освоить принцип перегрузки методов в Python (реализация через *overload* из `typing`).  
2. Научиться передавать аргументы различных типов и с разными значениями по умолчанию.  
3. Разработать простое приложение, демонстрирующее использование перегрузки и аргументов.  

## Теоретическая часть  
| Понятие | Описание |
|---------|----------|
| Перегрузка методов | Определение нескольких функций с одинаковым именем, но разным набором параметров. В Python реализуется через декоратор `@overload`. |
| `@overload` | Декоратор из модуля `typing`, который сообщает статическому анализатору о разных сигнатурах функции. |
| Конкретная реализация | В конце модуля размещается реальная функция без декоратора, обрабатывающая все варианты входных данных. |
| Аргументы по умолчанию | Параметр функции может иметь фиксированное значение, если его не передали. |
| Ключевые и позиционные аргументы | Позиционные – передаются по порядку; ключевые – задаются в виде `name=value`. |
| *Varargs* (`*args`) | Непредсказуемое количество позиционных аргументов. |
| *Kwargs* (`**kwargs`) | Непредсказуемое количество именованных аргументов. |
| `isinstance()` | Проверка типа объекта в рантайме. |
| Аннотации типов | `def func(a: int) -> str:` – описание типа входного и выходного параметров. |
| `typing.Protocol` | Позволяет описать «контракт» для объектов, реализующих определённый набор методов. |

## Ход выполнения работы  

1. **Создание проекта**  
   - Создайте папку `lab7` и файл `overload_demo.py`.  
   - Установите Python 3.11+ (для поддержки `typing.overload`).  

2. **Написание прототипов функций**  
   ```python
   from typing import overload, Union

   @overload
   def greet(name: str) -> str: ...
   
   @overload
   def greet(first_name: str, last_name: str) -> str: ...
   ```  
   Пояснение: два прототипа – один для одного аргумента, второй для двух.  

3. **Реализация конкретной функции**  
   ```python
   def greet(*args: str) -> str:
       if len(args) == 1:
           return f"Hello, {args[0]}!"
       elif len(args) == 2:
           return f"Hello, {args[0]} {args[1]}!"
       else:
           raise TypeError("greet() accepts 1 or 2 arguments")
   ```  
   Пояснение: функция обрабатывает оба случая, используя `*args`.  

4. **Проверка работы с аргументами по умолчанию**  
   ```python
   def add(a: int, b: int = 0) -> int:
       return a + b
   ```  
   Пример вызова: `add(5)` → `5`.  

5. **Использование *Varargs* и *Kwargs***  
   ```python
   def summarize(*values: int, **options: int) -> int:
       total = sum(values)
       for key, val in options.items():
           total += val
       return total
   ```  
   Пример: `summarize(1, 2, 3, bonus=10, penalty=2)` → `16`.  

6. **Тестирование**  
   ```python
   assert greet("Alice") == "Hello, Alice!"
   assert greet("Alice", "Smith") == "Hello, Alice Smith!"
   assert add(5) == 5
   assert summarize(1, 2, bonus=3) == 6
   ```  

7. **Запуск программы**  
   ```bash
   python overload_demo.py
   ```
