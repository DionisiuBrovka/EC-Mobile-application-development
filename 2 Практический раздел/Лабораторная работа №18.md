# Лабораторная работа № 18  
Разработка программ обработки файлов  

## Цель и задачи  
1. Освоить методы чтения, записи и обработки текстовых файлов в Python.  
2. Научиться работать с CSV‑форматом и библиотекой `pandas`.  
3. Разработать простое приложение, которое читает данные из файла, производит их агрегирование и сохраняет результат в новый файл.  

## Теоретическая часть  
| Понятие | Определение | Пример кода |
|---------|-------------|-------------|
| `open()` | Функция открывает файл и возвращает объект файла. | `f = open('file.txt', 'r')` |
| Режимы доступа | `'r'` – чтение, `'w'` – запись (создание/перезапись), `'a'` – добавление, `'b'` – бинарный режим. | `open('data.bin', 'wb')` |
| `with` | Контекстный менеджер, автоматически закрывает файл. | `with open('file.txt') as f:` |
| CSV | Формат разделённого запятой текста. | `"name,age\nAlice,30\nBob,25"` |
| `csv.reader` | Итератор, читающий строки CSV. | `csv.reader(f)` |
| `csv.writer` | Объект, записывающий строки CSV. | `csv.writer(f)` |
| `pandas.read_csv` | Читает CSV в DataFrame. | `df = pd.read_csv('data.csv')` |
| `DataFrame` | Табличная структура данных, поддерживающая операции группировки, фильтрации. | `df.groupby('city').mean()` |
| `to_csv` | Сохраняет DataFrame в CSV. | `df.to_csv('out.csv', index=False)` |
| `os.path` | Модуль для работы с путями файлов. | `os.path.exists('file.txt')` |
| `glob` | Поиск файлов по шаблону. | `glob.glob('*.txt')` |

## Ход выполнения работы  

1. **Подготовка исходного файла**  
   - Создайте текстовый файл `sales.txt` со следующим содержимым:  
     ```
     product,quantity,price
     apple,10,0.5
     banana,5,0.3
     orange,8,0.4
     ```
   - Пояснение: файл содержит заголовок и несколько строк с данными.  
   - Код создания (опционально):  
     ```python
     with open('sales.txt', 'w') as f:
         f.write('product,quantity,price\n')
         f.write('apple,10,0.5\n')
         f.write('banana,5,0.3\n')
         f.write('orange,8,0.4\n')
     ```

2. **Чтение файла строками**  
   - Откройте файл в режиме чтения и выведите все строки на экран.  
     ```python
     with open('sales.txt', 'r') as f:
         for line in f:
             print(line.strip())
     ```

3. **Парсинг CSV без сторонних библиотек**  
   - Используйте `csv.reader` для разбора данных.  
     ```python
     import csv
     with open('sales.txt', 'r') as f:
         reader = csv.reader(f)
         header = next(reader)
         data = [row for row in reader]
     print(header, data)
     ```

4. **Агрегирование данных**  
   - Вычислите общую выручку: `quantity * price` для каждой строки и сумму по всем строкам.  
     ```python
     total = sum(int(row[1]) * float(row[2]) for row in data)
     print('Total revenue:', total)
     ```

5. **Сохранение результата в новый CSV**  
   - Создайте файл `report.csv` с двумя колонками: `product,total_revenue`.  
     ```python
     with open('report.csv', 'w', newline='') as f:
         writer = csv.writer(f)
         writer.writerow(['product', 'total_revenue'])
         for row in data:
             revenue = int(row[1]) * float(row[2])
             writer.writerow([row[0], revenue])
     ```

6. **Использование pandas для упрощения**  
   - Перейдите к чтению и обработке через `pandas`.  
     ```python
     import pandas as pd
     df = pd.read_csv('sales.txt')
     df['revenue'] = df['quantity'] * df['price']
     df.to_csv('report_pandas.csv', columns=['product', 'revenue'], index=False)
     ```

7. **Экспериментальная часть**  
   - Измените исходный файл, добавив несколько строк.  
   - Запустите программу и сравните результаты в `report.csv` и `report_pandas.csv`.  
   - Задача: убедиться, что оба подхода дают одинаковый результат.
