# Лабораторная работа № 25  
Создание меню и диалоговых окон

## Цель и задачи
1. Научиться создавать графические интерфейсы с меню и диалоговыми окнами в Python.  
2. Овладеть библиотекой `tkinter` и её виджетами `Menu`, `messagebox`, `filedialog`.  
3. Понять принцип обработки событий в графическом приложении.  
4. Провести экспериментальное исследование поведения интерфейса при различных действиях пользователя.

## Теоретическая часть
| Понятие | Определение |
|---------|-------------|
| **Графический пользовательский интерфейс (GUI)** | Взаимодействие пользователя с программой через графические элементы. |
| **`tkinter`** | Стандартная библиотека Python для создания GUI. |
| **`Tk`** | Главный контейнер окна приложения. |
| **`Menu`** | Виджет, реализующий выпадающее меню. |
| **`Command`** | Атрибут команды, привязанный к пункту меню. |
| **`messagebox`** | Модуль для отображения стандартных диалоговых окон (info, warning, error). |
| **`filedialog`** | Модуль для диалогов выбора файлов и каталогов. |
| **Событие (event)** | Действие пользователя, которое приложение обрабатывает через callback‑функции. |
| **Callback** | Функция, вызываемая в ответ на событие. |
| **`pack`, `grid`, `place`** | Три менеджера размещения виджетов в `tkinter`. |

## Ход выполнения работы
1. **Инициализация окна**  
   ```python
   import tkinter as tk
   from tkinter import messagebox, filedialog

   root = tk.Tk()
   root.title("Меню и диалоги")
   root.geometry("400x300")
   ```
   *Пояснение:* создаём главное окно, задаём заголовок и размер.

2. **Создание главного меню**  
   ```python
   menubar = tk.Menu(root)
   root.config(menu=menubar)
   file_menu = tk.Menu(menubar, tearoff=0)
   menubar.add_cascade(label="Файл", menu=file_menu)
   ```
   *Пояснение:* `tearoff=0` отключает разделение меню.

3. **Добавление пунктов меню**  
   ```python
   def open_file():
       path = filedialog.askopenfilename()
       if path:
           messagebox.showinfo("Файл открыт", f"Путь: {path}")

   def exit_app():
       if messagebox.askyesno("Выход", "Вы действительно хотите выйти?"):
           root.destroy()

   file_menu.add_command(label="Открыть...", command=open_file)
   file_menu.add_separator()
   file_menu.add_command(label="Выход", command=exit_app)
   ```
   *Пояснение:* `open_file` открывает диалог выбора файла, `exit_app` подтверждает выход.

4. **Добавление другого меню (например, "Помощь")**  
   ```python
   help_menu = tk.Menu(menubar, tearoff=0)
   menubar.add_cascade(label="Помощь", menu=help_menu)

   def show_about():
       messagebox.showinfo("О программе", "Лабораторная работа №25\nСоздание меню и диалогов")

   help_menu.add_command(label="О программе", command=show_about)
   ```
   *Пояснение:* демонстрация использования диалога `info`.

5. **Запуск цикла обработки событий**  
   ```python
   root.mainloop()
   ```
   *Пояснение:* основной цикл, который ожидает и обрабатывает события.
