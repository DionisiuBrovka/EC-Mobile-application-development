## Реализация курсоров с асинхронными задачами

1. Освоить работу с асинхронными итераторами и курсорами в Python.  
2. Научиться реализовывать асинхронные генераторы для чтения больших файлов.

### Теоретическая часть
| Понятие | Определение |
|---------|-------------|
| Асинхронная функция | Функция, объявленная с `async def`, возвращающая объект `coroutine`. |

2. **Создание асинхронного генератора чтения файла**  
   Реализуем функцию `async_file_reader`, которая читает файл построчно в асинхронном режиме.  
   ```python
   import asyncio

   async def async_file_reader(path: str, buffer: int = 1024):
       """Асинхронный генератор чтения файла."""
       loop = asyncio.get_running_loop()
       with open(path, 'r', encoding='utf-8') as f:
           while True:
               data = await loop.run_in_executor(None, f.read, buffer)
               if not data:
                   break
               yield data
   ```

3. **Реализация асинхронного курсора**  
   Курсор будет оборачивать асинхронный генератор и предоставлять методы `next` и `close`.  
   ```python
   class AsyncCursor:
       def __init__(self, generator):
           self._gen = generator
           self._aiter = self._gen.__aiter__()

       async def next(self):
           try:
               return await self._aiter.__anext__()
           except StopAsyncIteration:
               return None

       async def close(self):
           if hasattr(self._gen, 'aclose'):
               await self._gen.aclose()
   ```

4. **Параллельная обработка данных**  
   Создадим несколько задач, каждая из которых будет читать и обрабатывать часть файла.  
   ```python
   async def process_chunk(cursor: AsyncCursor, chunk_id: int):
       while True:
           line = await cursor.next()
           if line is None:
               break
           # Пример обработки: подсчёт слов
           words = line.split()
           print(f"Chunk {chunk_id}: {len(words)} words")

   async def main():
       cursor = AsyncCursor(async_file_reader('large.txt'))
       tasks = [asyncio.create_task(process_chunk(cursor, i)) for i in range(4)]
       await asyncio.gather(*tasks)
       await cursor.close()

   asyncio.run(main())
   ```
