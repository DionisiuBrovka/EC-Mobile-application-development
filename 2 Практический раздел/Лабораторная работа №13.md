# Лабораторная работа № 13  
**Разработка программ обработки строк**

## Цель и задачи

1. Ознакомиться с основными методами работы со строками в Python.  
2. Научиться использовать регулярные выражения для поиска и замены шаблонов.  
3. Разработать простые программы, выполняющие операции над строками: подсчёт, поиск, фильтрация, форматирование.  
4. Оценить эффективность различных подходов к обработке строк.

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| Строка (string) | Последовательность символов, заключённая в кавычки. |
| Элемент строки | Любой символ, включая пробелы, цифры, знаки препинания. |
| Индекс | Позиция символа в строке, начинается с 0. |
| Срез (slice) | Подстрока, получаемая как `s[start:stop:step]`. |
| Методы строки | `len`, `lower`, `upper`, `strip`, `replace`, `split`, `join`, `find`, `index`, `count`, `format`, `startswith`, `endswith`. |
| Регулярные выражения (regex) | Набор символов, описывающих шаблон поиска в строке. |
| Метасимволы | `.` – любой символ, `^` – начало строки, `$` – конец строки, `*` – 0+ повторений, `+` – 1+ повторений, `?` – 0+ (нежадно), `[]` – набор символов, `()` – группа. |
| Функция `re.search()` | Возвращает объект совпадения, если шаблон найден. |
| Функция `re.sub()` | Заменяет все вхождения шаблона на заданную строку. |
| Эффективность | Встроенные методы строки обычно быстрее, чем ручные циклы. |

## Ход выполнения работы

1. **Подготовка окружения**  
   * Установить Python 3.x (если ещё не установлен).  
   * Создать новый файл `string_processing.py`.  

2. **Определить набор задач**  
   * Подсчёт количества слов в тексте.  
   * Замена всех вхождений слова «Python» на «Python‑3».  
   * Удаление всех чисел из строки.  
   * Форматирование даты из `YYYYMMDD` в `DD.MM.YYYY`.  

3. **Реализация функций**  
   ```python
   import re

   def count_words(text: str) -> int:
       """Возвращает количество слов в строке."""
       words = text.split()
       return len(words)

   def replace_python(text: str) -> str:
       """Заменяет все вхождения 'Python' на 'Python‑3'."""
       return re.sub(r'\bPython\b', 'Python‑3', text)

   def remove_numbers(text: str) -> str:
       """Удаляет все цифры из строки."""
       return re.sub(r'\d+', '', text)

   def format_date(date_str: str) -> str:
       """Форматирует строку даты из YYYYMMDD в DD.MM.YYYY."""
       if len(date_str) != 8 or not date_str.isdigit():
           raise ValueError('Неверный формат даты')
       return f'{date_str[6:8]}.{date_str[4:6]}.{date_str[0:4]}'
   ```

4. **Тестирование функций**  
   ```python
   if __name__ == "__main__":
       sample = "Python 3.10 released on 20211004. Numbers: 12345."
       print(count_words(sample))          # 7
       print(replace_python(sample))       # "Python‑3 3.10 released on 20211004. Numbers: 12345."
       print(remove_numbers(sample))      # "Python 3.10 released on . Numbers: ."
       print(format_date("20211004"))      # "04.10.2021"
   ```

5. **Анализ результатов**  
   * Сравнить время выполнения функций с помощью `timeit`.  
   * Оценить различия между использованием регулярных выражений и простыми строковыми методами.
