# Лабораторная работа № 6  
Разработка методов и использование их в программах

## Цель и задачи

1. Научиться проектировать и реализовывать собственные функции (методы) в Python.  
2. Понять принципы передачи аргументов, возврата значений и работы с глобальными переменными.  
3. Освоить технику рефакторинга кода: вынесение повторяющихся блоков в отдельные методы.  
4. Оценить влияние модульности на читаемость, тестируемость и поддержку программы.

## Теоретическая часть

| Понятие | Описание |
|--------|----------|
| **Функция** | Самостоятельный блок кода, выполняющий конкретную задачу и вызываемый по имени. |
| **Параметр** | Именованная переменная, принимаемая функцией при вызове. |
| **Аргумент** | Конкретное значение, передаваемое в функцию. |
| **Возврат (return)** | Операция, завершающая выполнение функции и передающая результат. |
| **Глобальная переменная** | Переменная, объявленная вне функций, доступная внутри них при использовании `global`. |
| **Локальная переменная** | Переменная, объявленная внутри функции, недоступна снаружи. |
| **Докстринг** | Строка, описывающая работу функции; доступна через `help()`. |
| **Ключевые слова `*args` и `**kwargs`** | Позволяют принимать произвольное количество позиционных/ключевых аргументов. |
| **Модуль** | Файл с расширением `.py`, содержащий функции, классы и переменные. |
| **Импорт** | Включение кода из другого модуля через `import` или `from … import …`. |

## Ход выполнения работы

1. **Создание проекта**  
   - Создайте папку `lab6` и в ней файл `main.py`.  
   - В начале файла добавьте комментарий с описанием лабораторной работы.

2. **Разработка базовой функции**  
   - Напишите функцию `def greet(name: str) -> str:`, которая возвращает строку «Привет, {name}!».  
   - Добавьте docstring и типизацию.  
   - Пример использования:
     ```python
     print(greet("Алексей"))  # Привет, Алексей!
     ```

3. **Параметры по умолчанию**  
   - Расширьте `greet`, добавив необязательный параметр `greeting: str = "Привет"`.  
   - Функция должна работать как `greet("Иван")` и как `greet("Иван", greeting="Здравствуйте")`.

4. **Аргументы переменной длины**  
   - Создайте функцию `def concat(*args: str) -> str:` – объединяет произвольное количество строк через пробел.  
   - Демонстрация:
     ```python
     print(concat("Python", "это", "круто"))  # Python это круто
     ```

5. **Работа с глобальными переменными**  
   - Объявите глобальную переменную `counter = 0`.  
   - Напишите функцию `def increment() -> None:` которая увеличивает `counter` на 1.  
   - Используйте ключевое слово `global` внутри функции.

6. **Модульность**  
   - Вынесите функции `greet`, `concat` и `increment` в отдельный файл `utils.py`.  
   - В `main.py` импортируйте их через `from utils import greet, concat, increment`.

7. **Тестирование**  
   - Напишите простые тесты в файле `test_utils.py` с помощью `unittest`.  
   - Проверьте корректность работы функций, включая граничные случаи.

8. **Документация**  
   - Сгенерируйте автодокумент с помощью `pydoc` или `mkdocs`.  
   - Включите в отчёт ссылки на сгенерированную документацию (файл `README.md`).

9. **Экспериментальная часть**  
   - Реализуйте простую CLI‑программу, которая использует функции из `utils.py`.  
   - Попросите студентов запускать программу с разными аргументами и фиксировать результаты.

## Контрольные вопросы / задания для самопроверки

1. Что такое глобальная переменная и как её изменить внутри функции?  
2. Напишите функцию `factorial(n: int) -> int`, использующую рекурсию.  
3. Как передать в функцию произвольное количество именованных аргументов?  
4. В чем разница между `return` и `print` в функции?  
5. Как импортировать только одну функцию из модуля?  
6. Что произойдет, если вызвать `increment()` без ключевого слова `global`?  
7. Опишите преимущества вынесения кода в отдельный модуль.  
8. Напишите тест для функции `concat`, проверяющий, что она корректно обрабатывает пустой список аргументов.  
9. Как задать тип возвращаемого значения функции, которая ничего не возвращает?  
10. Какие преимущества предоставляет использование docstring в функции?
