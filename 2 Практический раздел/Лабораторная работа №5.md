# Лабораторная работа № 5  
Разработка классов и использование их в программах

## Цель и задачи
1. Ознакомиться с принципами объектно‑ориентированного программирования в Python.  
2. Научиться определять классы, атрибуты и методы, а также использовать их в программах.  
3. Понять роль конструкторов (`__init__`) и магических методов (`__str__`, `__repr__`, `__add__` и др.).  
4. Реализовать простое приложение, использующее созданные классы, и проанализировать его работу.

## Теоретическая часть
| Понятие | Описание |
|---------|----------|
| Класс | Шаблон, описывающий тип данных (структуру и поведение). |
| Объект | Экземпляр класса, имеющий собственные данные. |
| Атрибут | Переменная, принадлежащая классу или объекту. |
| Метод | Функция, определенная внутри класса, применяемая к объекту. |
| Конструктор | Специальный метод `__init__`, вызываемый при создании объекта. |
| Инкапсуляция | Сокрытие внутренней реализации класса от внешнего кода. |
| Наследование | Механизм, позволяющий создавать новый класс на основе существующего. |
| Полиморфизм | Способность объектов разных классов обрабатывать одинаковые сообщения. |
| Магический метод | Специальный метод, имя которого начинается и заканчивается двойным подчёркиванием. |
| Декоратор | Функция, принимающая другую функцию или класс и возвращающая модифицированную версию. |

## Ход выполнения работы
1. **Создание проекта**  
   - Создайте папку `lab5` и внутри неё два файла: `main.py` и `shapes.py`.  
   - В `shapes.py` определите абстрактный класс `Shape` с методом `area()` (используйте `abc` модуль).  

   ```python
   # shapes.py
   from abc import ABC, abstractmethod

   class Shape(ABC):
       @abstractmethod
       def area(self) -> float:
           pass
   ```

2. **Определение классов фигур**  
   - Наследуйте `Circle`, `Rectangle` и `Triangle` от `Shape`.  
   - Реализуйте конструкторы, атрибуты (`radius`, `width`, `height`, `base`, `height`) и метод `area()`.  

   ```python
   # shapes.py (дальше)
   import math

   class Circle(Shape):
       def __init__(self, radius: float):
           self.radius = radius

       def area(self) -> float:
           return math.pi * self.radius ** 2

   class Rectangle(Shape):
       def __init__(self, width: float, height: float):
           self.width = width
           self.height = height

       def area(self) -> float:
           return self.width * self.height

   class Triangle(Shape):
       def __init__(self, base: float, height: float):
           self.base = base
           self.height = height

       def area(self) -> float:
           return 0.5 * self.base * self.height
   ```

3. **Магические методы**  
   - Добавьте `__str__` к каждому классу для красивого вывода.  

   ```python
   # shapes.py (дальше)
   class Circle(Shape):
       # ... предыдущий код
       def __str__(self):
           return f"Circle(radius={self.radius})"

   # аналогично для Rectangle и Triangle
   ```

4. **Использование классов в главной программе**  
   - В `main.py` создайте список фигур, заполните его объектами разных классов.  
   - Выведите информацию о каждой фигуре и её площади.  

   ```python
   # main.py
   from shapes import Circle, Rectangle, Triangle

   def main():
       shapes = [
           Circle(5),
           Rectangle(4, 6),
           Triangle(3, 7)
       ]

       for shape in shapes:
           print(f"{shape}: area = {shape.area():.2f}")

   if __name__ == "__main__":
       main()
   ```

5. **Экспериментальная часть**  
   - Измените параметры фигур, добавьте новые типы (например, `Square`).  
   - Проверьте, что методы работают корректно и вывод корректен.  

6. **Анализ результатов**  
   - Сравните вычисленные площади с теоретическими значениями.  
   - Обсудите, как изменение атрибутов влияет на поведение объектов.
