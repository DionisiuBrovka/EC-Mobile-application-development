# Лабораторная работа № 10  
Разработка программ, реализующих и использующих интерфейсы  

## Цель и задачи  

1. Изучить принципы проектирования и реализации пользовательских интерфейсов (GUI) на Python.  
2. Освоить работу с популярными библиотеками для создания графических приложений (`tkinter`, `PyQt5`/`PySide2`).  
3. Научиться объединять логику программы и визуальный компонент через модели и контроллеры.  
4. Разработать простое приложение, демонстрирующее взаимодействие интерфейса с данными (CRUD‑операции).  

## Теоретическая часть  

1. **Интерфейс пользователя (UI)** – набор визуальных элементов, с помощью которых пользователь взаимодействует с программой.  
2. **Модель–Вид–Контроллер (MVC)** – паттерн проектирования, разделяющий логику программы, представление и управление.  
3. **Сигналы и слоты** – механизм связи компонентов в `Qt`, позволяющий реагировать на события.  
4. **Пул потоков (ThreadPoolExecutor)** – средство для выполнения фоновых задач, не блокирующих UI.  
5. **Событийный цикл** – непрерывный цикл, в котором обрабатываются пользовательские события и обновления графики.  
6. **Класс `QWidget`** – базовый класс для всех виджетов в `Qt`.  
7. **Класс `tkinter.Tk`** – главный контейнер, в котором размещаются остальные виджеты.  
8. **Метод `pack()`, `grid()`, `place()`** – способы размещения виджетов в `tkinter`.  
9. **Класс `QMainWindow`** – основное окно приложения с меню, статусной строкой и центральным виджетом.  
10. **Паттерн «Фасад»** – упрощает взаимодействие с сложной подсистемой интерфейса.  

## Ход выполнения работы  

1. **Подготовка рабочего пространства**  
   - Установить Python 3.11+ и пакет `pip install pyqt5` (или `pyside2`).  
   - Создать проект `lab10` и каталог `src/`.  

2. **Создание главного окна**  
   ```python
   # src/main_window.py
   from PyQt5.QtWidgets import QMainWindow, QApplication, QWidget, QVBoxLayout, QPushButton
   import sys

   class MainWindow(QMainWindow):
       def __init__(self):
           super().__init__()
           self.setWindowTitle("Пример GUI")
           central = QWidget()
           layout = QVBoxLayout()
           btn = QPushButton("Нажми меня")
           layout.addWidget(btn)
           central.setLayout(layout)
           self.setCentralWidget(central)

   if __name__ == "__main__":
       app = QApplication(sys.argv)
       win = MainWindow()
       win.show()
       sys.exit(app.exec_())
   ```  
   Пояснение: создаём окно, добавляем кнопку и выводим его.  

3. **Разработка модели данных**  
   - Создать класс `Item` с атрибутами `id`, `name`, `value`.  
   - Реализовать класс `ItemRepository`, хранящий список объектов и методы `add`, `remove`, `update`, `list`.  

4. **Интеграция модели с UI**  
   - Добавить виджет `QTableView` в главное окно.  
   - Использовать `QStandardItemModel` для отображения данных из репозитория.  
   - Реализовать слоты для кнопок «Добавить», «Удалить», «Обновить».  

5. **Обработка событий**  
   - Связать сигнал `clicked` кнопок с методами контроллера.  
   - Внутри методов обращаться к репозиторию и обновлять модель таблицы.  

6. **Фоновая работа**  
   - При выполнении долгих операций (например, загрузка данных) использовать `QThreadPool` и `QRunnable`.  
   - Показать индикатор выполнения (`QProgressBar`).  

7. **Тестирование**  
   - Запустить приложение, проверить корректность CRUD‑операций.  
   - Убедиться, что UI остаётся отзывчивым при фоновых задачах.  

8. **Подготовка отчёта**  
   - Сформировать текстовый файл `report.md` (см. ниже).

Студент должен подготовить Markdown‑файл `report.md` со следующей структурой:  

1. **Введение** – цель лабораторной работы.  
2. **Теоретическая часть** – краткое изложение понятий, использованных в работе.  
3. **Описание реализованного решения**  
   - Архитектура программы (описание MVC).  
   - Краткие коды ключевых классов (`MainWindow`, `Item`, `ItemRepository`).  
4. **Экспериментальная часть** – результаты тестирования:

5. **Выводы** – что было достигнуто, какие навыки закреплены.  
6. **Приложения** – исходный код (полные файлы), screenshots интерфейса.
