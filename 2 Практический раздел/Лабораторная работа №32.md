# Лабораторная работа № 32  
**Использование базы данных SQLite**

## Цель и задачи  
1. Познакомиться с принципами работы реляционных баз данных и системой SQLite.  
2. Научиться создавать, изменять и удалять таблицы, а также выполнять операции CRUD (Create‑Read‑Update‑Delete).  
3. Освоить работу с библиотекой `sqlite3` в Python: подключение к базе, подготовленные запросы, транзакции.  
4. Проанализировать результаты работы программы и оценить эффективность выбранного подхода.

## Теоретическая часть  
| Понятие | Определение |
|---------|-------------|
| **База данных** | Коллекция данных, организованных логически и доступных через систему управления базами данных (СУБД). |
| **Таблица** | Структура данных, состоящая из строк (записей) и столбцов (полей). |
| **Запись (строка)** | Набор значений, соответствующих каждому столбцу таблицы. |
| **Первичный ключ** | Поле (или набор полей), уникально идентифицирующее каждую запись в таблице. |
| **Внешний ключ** | Поле, ссылающееся на первичный ключ другой таблицы, обеспечивающее целостность данных. |
| **SQL** | Язык структурированных запросов – язык описания и манипуляции данными. |
| **CRUD** | Операции создания (Create), чтения (Read), обновления (Update) и удаления (Delete) данных. |
| **Транзакция** | Последовательность операций, выполняемых атомарно: либо все успешно, либо откатываются. |
| **Подготовленный запрос** | SQL‑запрос с параметрами, который компилируется один раз и может быть выполнен многократно. |
| **`sqlite3`** | Встроенная библиотека Python для работы с SQLite‑базой. |

## Ход выполнения работы  

1. **Создание проекта и подключение к базе**  
   ```python
   import sqlite3

   conn = sqlite3.connect('students.db')   # создаёт файл базы, если его нет
   cursor = conn.cursor()
   ```
   *Пояснение*: `connect()` открывает соединение, `cursor()` – объект для выполнения SQL‑команд.

2. **Создание таблиц**  
   ```python
   cursor.execute("""
       CREATE TABLE IF NOT EXISTS departments (
           id INTEGER PRIMARY KEY AUTOINCREMENT,
           name TEXT NOT NULL
       )
   """)
   cursor.execute("""
       CREATE TABLE IF NOT EXISTS students (
           id INTEGER PRIMARY KEY AUTOINCREMENT,
           first_name TEXT NOT NULL,
           last_name TEXT NOT NULL,
           department_id INTEGER,
           FOREIGN KEY(department_id) REFERENCES departments(id)
       )
   """)
   conn.commit()
   ```
   *Пояснение*: `IF NOT EXISTS` предотвращает ошибку при повторном запуске. `AUTOINCREMENT` генерирует уникальные ключи.

3. **Вставка данных (Create)**  
   ```python
   cursor.execute("INSERT INTO departments (name) VALUES (?)", ("Computer Science",))
   cursor.execute("INSERT INTO departments (name) VALUES (?)", ("Mathematics",))
   cursor.execute("""
       INSERT INTO students (first_name, last_name, department_id)
       VALUES (?, ?, ?)
   """, ("Ivan", "Ivanov", 1))
   conn.commit()
   ```
   *Пояснение*: Параметризованные запросы защищают от SQL‑инъекций.

4. **Запрос данных (Read)**  
   ```python
   cursor.execute("""
       SELECT s.id, s.first_name, s.last_name, d.name
       FROM students s
       LEFT JOIN departments d ON s.department_id = d.id
   """)
   for row in cursor.fetchall():
       print(row)
   ```
   *Пояснение*: `LEFT JOIN` обеспечивает вывод всех студентов даже без отдела.

5. **Обновление данных (Update)**  
   ```python
   cursor.execute("""
       UPDATE students
       SET department_id = ?
       WHERE last_name = ?
   """, (2, "Ivanov"))
   conn.commit()
   ```
   *Пояснение*: Перемещаем студента в другой отдел.

6. **Удаление данных (Delete)**  
   ```python
   cursor.execute("DELETE FROM students WHERE id = ?", (1,))
   conn.commit()
   ```
   *Пояснение*: Удаляем запись по первичному ключу.

7. **Транзакции и откат**  
   ```python
   try:
       cursor.execute("INSERT INTO students (first_name, last_name) VALUES (?, ?)", ("Anna", "Smith"))
       conn.commit()
   except sqlite3.Error:
       conn.rollback()
   ```
   *Пояснение*: Если запрос завершится ошибкой, изменения откатываются.

8. **Закрытие соединения**  
   ```python
   cursor.close()
   conn.close()
   ```
