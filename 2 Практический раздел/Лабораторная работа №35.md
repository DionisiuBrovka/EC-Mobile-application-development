## Теоретическая часть

| Понятие | Описание |
|---------|----------|
| **Фоновая служба** | Процесс, выполняющийся независимо от пользовательского интерфейса, часто в виде потока (`threading.Thread`) или процесса (`multiprocessing.Process`). |
| **Поток (Thread)** | Легковесный процесс, разделяющий память с главным потоком; используется для параллельного выполнения задач. |
| **Процесс** | Самостоятельная единица исполнения с собственным пространством памяти. |
| **Системное уведомление** | Всплывающее сообщение, которое отображается в панели задач/уведомлений ОС. |
| **Библиотека `plyer`** | Кроссплатформенный API для доступа к системным функциям (уведомления, GPS, датчик батареи и др.). |
| **JSON** | Формат обмена данными, основанный на текстовом представлении объектов `dict`/`list`. |
| **ConfigParser** | Модуль для работы с INI‑файлами, поддерживает секции и ключ‑значение. |
| **`@property`** | Декоратор, позволяющий обращаться к методу как к атрибуту. |
| **`__init__`** | Конструктор класса, вызываемый при создании объекта. |
| **`__del__`** | Деструктор, вызываемый при уничтожении объекта (не гарантирует момент выполнения). |
| **`json.dump()` / `json.load()`** | Функции для сериализации/десериализации объектов в/из JSON. |

## Ход выполнения работы

1. **Подготовка окружения**  
   Установить необходимые библиотеки.  
   ```bash
   pip install plyer
   ```

2. **Создание класса `AppSettings`**  
   Обеспечить загрузку/сохранение настроек в JSON.  
   ```python
   import json
   from pathlib import Path

   class AppSettings:
       def __init__(self, file_path='settings.json'):
           self.file_path = Path(file_path)
           self.data = {}
           self.load()

       def load(self):
           if self.file_path.exists():
               with self.file_path.open('r', encoding='utf-8') as f:
                   self.data = json.load(f)

       def save(self):
           with self.file_path.open('w', encoding='utf-8') as f:
               json.dump(self.data, f, indent=4)

       def __getitem__(self, key):
           return self.data.get(key)

       def __setitem__(self, key, value):
           self.data[key] = value
   ```

3. **Разработка фоновой службы**  
   Сервис будет выполнять простую задачу (например, обновлять таймер) и отправлять уведомление.  
   ```python
   import threading
   import time
   from plyer import notification

   class BackgroundService(threading.Thread):
       def __init__(self, interval=10):
           super().__init__(daemon=True)
           self.interval = interval
           self.running = False

       def run(self):
           self.running = True
           while self.running:
               time.sleep(self.interval)
               self.notify()

       def notify(self):
           notification.notify(
               title='Фоновая служба',
               message=f'Служба выполнила работу каждые {self.interval} сек.',
               timeout=5
           )

       def stop(self):
           self.running = False
   ```

4. **Интеграция с пользовательским интерфейсом**  
   Для простоты используем консольный интерфейс.  
   ```python
   def main():
       settings = AppSettings()
       service_interval = settings['interval'] or 10
       service = BackgroundService(interval=service_interval)
       service.start()

       try:
           while True:
               cmd = input('> ').strip().lower()
               if cmd == 'exit':
                   break
               elif cmd.startswith('set '):
                   _, key, value = cmd.split()
                   settings[key] = int(value)
                   settings.save()
                   print(f'Настройка {key} установлена в {value}')
               else:
                   print('Команды: set <key> <value>, exit')
       finally:
           service.stop()
           service.join()
   ```

5. **Тестирование**  
   Запустить приложение, проверить:
   - правильность сохранения настроек;
   - корректную работу фоновой службы;
   - отображение уведомлений.
