# Лабораторная работа № 31  
**Реализация выдвижных панелей**

---

## Цель и задачи

1. Научиться проектировать и реализовывать выдвижные панели в графическом интерфейсе.
2. Освоить работу с модулем `tkinter` и его виджетами `PanedWindow`, `Frame`, `Button`.
3. Научиться управлять состоянием виджетов (включение/выключение, скрытие/показ).
4. Оценить влияние параметров панели (ширина, высота, направление выдвижения) на пользовательский интерфейс.

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| Выдвижная панель | Виджет, который может скрываться и появляться по запросу пользователя. |
| `PanedWindow` | Виджет `tkinter`, объединяющий несколько панелей в одну. |
| `Frame` | Базовый контейнер для размещения других виджетов. |
| `pack()` | Метод размещения виджетов по вертикали/горизонтали. |
| `grid()` | Метод размещения виджетов по сетке. |

| `lift()` / `lower()` | Методы управления порядком наложения виджетов. |
| `geometry()` | Установка размеров окна. |
| `mainloop()` | Главный цикл обработки событий. |
| `lambda` | Анонимная функция, часто используемая в `command` для виджетов. |
| `state` | Атрибут виджета, определяющий его активность (`normal`, `disabled`). |

## Ход выполнения работы

1. **Создание основного окна**  
   ```python
   import tkinter as tk
   root = tk.Tk()
   root.title("Выдвижные панели")
   root.geometry("600x400")
   ```

2. **Размещение панели с кнопкой управления**  
   ```python
   top_frame = tk.Frame(root, bg="lightgray")
   top_frame.pack(side="top", fill="x")
   toggle_btn = tk.Button(top_frame, text="Показать панель")
   toggle_btn.pack(side="right", padx=5, pady=5)
   ```

3. **Создание выдвижной панели**  
   ```python
   side_panel = tk.Frame(root, bg="white", width=200, height=400)
   side_panel.pack(side="left", fill="y")
   side_panel.pack_forget()  # Скрыть изначально
   ```

4. **Функция переключения панели**  
   ```python
   def toggle_panel():
       if side_panel.winfo_ismapped():
           side_panel.pack_forget()
           toggle_btn.config(text="Показать панель")
       else:
           side_panel.pack(side="left", fill="y")
           toggle_btn.config(text="Скрыть панель")
   toggle_btn.config(command=toggle_panel)
   ```

5. **Добавление контента в панель**  
   ```python
   lbl = tk.Label(side_panel, text="Контент панели", bg="white")
   lbl.pack(padx=10, pady=10)
   ```

6. **Запуск главного цикла**  
   ```python
   root.mainloop()
   ```

7. **Эксперимент**  
   - Измените размеры панели (`width`, `height`).  
   - Попробуйте разместить панель справа, снизу.  
   - Добавьте несколько панелей и управляйте ими отдельными кнопками.
