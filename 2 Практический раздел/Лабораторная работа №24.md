# Лабораторная работа № 24  
Создание виджетов‑списков

## Цель и задачи  
1. Научиться создавать и настраивать виджеты‑списки в графическом интерфейсе Python.  
2. Освоить работу с элементами управления `Listbox`, `Combobox` и `Treeview`.  
3. Понять механизм обработки событий, связанных с выбором элементов списка.  
4. Разработать простое приложение, демонстрирующее взаимодействие списка с другими виджетами.

## Теоретическая часть  
| Понятие | Определение |
|---------|-------------|
| **Widget (виджет)** | Самостоятельный компонент графического интерфейса, реализующий определённый функционал (кнопка, поле ввода, список и т.д.). |
| **Listbox** | Виджет, отображающий вертикальный список строк с возможностью выделения одного или нескольких элементов. |
| **Combobox** | Виджет, объединяющий выпадающий список и поле ввода; позволяет выбирать элемент из списка или вводить произвольный текст. |
| **Treeview** | Виджет древовидной структуры, где каждый узел может иметь дочерние узлы; часто используется для отображения иерархических данных. |
| **Event Binding** | Привязка обработчика события (например, `<<ListboxSelect>>`) к виджету для реагирования на пользовательские действия. |
| **`insert()`** | Метод, добавляющий элемент в `Listbox`/`Treeview` по указанному индексу. |
| **`get()`** | Метод, возвращающий выбранный элемент списка. |
| **`delete()`** | Метод, удаляющий элемент из списка по индексу. |
| **`selection_set()`** | Метод, программно устанавливающий выбранный элемент. |
| **`pack()` / `grid()` / `place()`** | Методы размещения виджетов внутри контейнера. |

## Ход выполнения работы  

1. **Создание окна приложения**  
   ```python
   import tkinter as tk
   from tkinter import ttk

   root = tk.Tk()
   root.title("Виджеты‑списки")
   root.geometry("400x300")
   ```

2. **Добавление `Listbox`**  
   ```python
   listbox = tk.Listbox(root, height=8, width=30)
   listbox.pack(pady=10)
   # Заполнение списка
   items = ["Пункт 1", "Пункт 2", "Пункт 3", "Пункт 4"]
   for item in items:
       listbox.insert(tk.END, item)
   ```

3. **Обработчик выбора в `Listbox`**  
   ```python
   def on_select(event):
       selection = listbox.curselection()
       if selection:
           index = selection[0]
           value = listbox.get(index)
           label.config(text=f"Вы выбрали: {value}")

   listbox.bind('<<ListboxSelect>>', on_select)
   ```

4. **Добавление `Combobox`**  
   ```python
   combo = ttk.Combobox(root, values=items, state='readonly')
   combo.pack(pady=5)
   combo.set("Выберите элемент")
   ```

5. **Обработчик `Combobox`**  
   ```python
   def on_combo(event):
       value = combo.get()
       label.config(text=f"Вы выбрали: {value}")

   combo.bind('<<ComboboxSelected>>', on_combo)
   ```

6. **Создание `Treeview`**  
   ```python
   tree = ttk.Treeview(root, columns=("info"), show='tree')
   tree.pack(pady=10, fill='x')
   tree.insert('', 'end', text='Корень', iid='root')
   tree.insert('root', 'end', text='Дочерний 1', iid='child1')
   tree.insert('root', 'end', text='Дочерний 2', iid='child2')
   ```

7. **Обработчик выбора в `Treeview`**  
   ```python
   def on_tree_select(event):
       selected = tree.selection()
       if selected:
           value = tree.item(selected[0], 'text')
           label.config(text=f"Вы выбрали: {value}")

   tree.bind('<<TreeviewSelect>>', on_tree_select)
   ```

8. **Метка для вывода результата**  
   ```python
   label = tk.Label(root, text="Выберите элемент из списка")
   label.pack(pady=5)
   ```

9. **Запуск цикла обработки событий**  
   ```python
   root.mainloop()
   ```

## Контрольные вопросы / задания для самопроверки  

1. Что делает метод `curselection()` у виджета `Listbox`?  
2. Как изменить состояние `Combobox` так, чтобы пользователь мог вводить произвольный текст?  
3. В чем разница между `pack()`, `grid()` и `place()` при размещении виджетов?  
4. Как программно удалить третий элемент из `Listbox`?  
5. Какие события можно привязать к виджету `Treeview` для обработки выбора узла?  
6. Напишите код, который добавит в `Treeview` новый узел с текстом "Новый узел" под элементом "Дочерний 1".  
7. Как изменить высоту списка `Listbox` без изменения количества строк?  
8. Что произойдёт, если вызвать `delete(0, tk.END)` для `Listbox`?  
9. Как получить все выбранные элементы в `Listbox` при включённом режиме множественного выбора?  
10. Создайте функцию, которая будет выводить в `label` список всех элементов, выбранных в `Listbox`.
