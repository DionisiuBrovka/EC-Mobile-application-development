# Лабораторная работа № 22  
Разработка пользовательского интерфейса

## Цель и задачи
1. Научиться проектировать и реализовывать графический пользовательский интерфейс (GUI) на Python.  
2. Освоить основные элементы библиотеки Tkinter: окна, виджеты, менеджеры размещения.  
3. Разработать простое приложение с интерактивным вводом и выводом данных.  
4. Провести экспериментальную проверку корректности работы интерфейса и проанализировать полученные результаты.

## Теоретическая часть
1. **GUI (Graphical User Interface)** – графический пользовательский интерфейс, позволяющий пользователю взаимодействовать с программой с помощью графических элементов.  
2. **Tkinter** – стандартный модуль Python для создания GUI, основанный на Tcl/Tk.  
3. **Widget** – виджет, элемент интерфейса (кнопка, метка, поле ввода и др.).  
4. **Geometry Manager** – менеджер размещения виджетов: `pack`, `grid`, `place`.  
5. **Event Loop** – цикл обработки событий, в котором приложение ожидает пользовательских действий.  
6. **Callback** – функция‑обработчик, вызываемая при событии (нажатие кнопки и т.п.).  
7. **Layout** – расположение виджетов в окне.  
8. **Frame** – контейнер для группировки виджетов.  
9. **StringVar, IntVar** – переменные, привязанные к виджетам, обеспечивающие синхронизацию данных.  
10. **Mainloop()** – метод, запускающий цикл обработки событий.

## Ход выполнения работы
1. **Создание основного окна**  
   ```python
   import tkinter as tk

   root = tk.Tk()
   root.title("Пример GUI")
   root.geometry("300x200")
   ```
   *Пояснение:* создаём объект `Tk`, задаём заголовок и размер окна.

2. **Добавление виджетов**  
   - Метка (Label)  
   - Поле ввода (Entry)  
   - Кнопка (Button)  
   ```python
   lbl = tk.Label(root, text="Введите число:")
   lbl.pack(pady=10)

   entry = tk.Entry(root)
   entry.pack(pady=5)

   btn = tk.Button(root, text="Показать квадрат")
   btn.pack(pady=10)
   ```
   *Пояснение:* виджеты размещаются с помощью `pack` и отступов.

3. **Определение обработчика события**  
   ```python
   def show_square():
       try:
           num = float(entry.get())
           result = num ** 2
           result_lbl.config(text=f"Квадрат: {result}")
       except ValueError:
           result_lbl.config(text="Ошибка: введите число")

   btn.config(command=show_square)
   ```
   *Пояснение:* функция получает ввод, вычисляет квадрат и обновляет метку.

4. **Добавление метки для вывода результата**  
   ```python
   result_lbl = tk.Label(root, text="")
   result_lbl.pack(pady=5)
   ```
   *Пояснение:* метка будет обновляться при каждом нажатии кнопки.

5. **Запуск цикла обработки событий**  
   ```python
   root.mainloop()
   ```
   *Пояснение:* приложение ждет пользовательских действий.

6. **Экспериментальная проверка**  
   - Ввод корректных чисел (целых, дробных).  
   - Ввод некорректных данных (строк, символов).  
   - Проверка, что окно корректно реагирует на события и выводит правильный результат.
