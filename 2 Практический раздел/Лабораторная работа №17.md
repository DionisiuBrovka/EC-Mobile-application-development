# Лабораторная работа № 17  
Разработка программ создания файлов  

## Цель и задачи  
1. Изучить основные принципы работы с файловой системой в Python.  
2. Освоить функции модуля `os` и `pathlib` для создания, удаления и проверки существования файлов и каталогов.  
3. Научиться обрабатывать ошибки при работе с файлами (исключения `FileNotFoundError`, `PermissionError` и др.).  
4. Разработать небольшое консольное приложение, создающее структуру каталогов и файлов по заданному шаблону.  

## Теоретическая часть  
| Понятие | Определение |
|---------|-------------|
| Файл | Бинарный или текстовый объект хранения данных, идентифицируемый путём. |
| Путь | Строка, указывающая на расположение файла/каталога в файловой системе. |
| Абсолютный путь | Полный путь от корневой директории до объекта. |
| Относительный путь | Путь, заданный относительно текущего рабочего каталога. |
| Модуль `os` | Стандартный модуль для взаимодействия с операционной системой (создание/удаление каталогов, проверка существования файлов). |
| Модуль `pathlib` | Классический объектно‑ориентированный интерфейс к путям (путь как объект). |
| Исключение `FileNotFoundError` | Генерируется, если заданный файл/каталог не найден. |
| Исключение `PermissionError` | Генерируется при попытке доступа к файлу/каталогу без прав. |
| Функция `open()` | Открывает файл и возвращает файловый объект. |
| Режим открытия `w` | Создаёт новый файл или перезаписывает существующий. |
| Режим открытия `x` | Создаёт новый файл, генерирует ошибку, если файл существует. |

## Ход выполнения работы  

1. **Подготовка рабочего окружения**  
   - Создайте новый каталог `lab17` в домашней директории.  
   ```python
   import os
   os.makedirs('lab17', exist_ok=True)
   ```
   - Убедитесь, что каталог создан, используя `os.path.isdir`.  

2. **Создание файловой структуры**  
   - Определите шаблон: в каталоге `lab17` создаются подпапки `docs`, `data` и `src`.  
   - Для каждой подпапки создайте файл `README.md` с содержимым «Пустой каталог».  
   ```python
   from pathlib import Path

   base = Path('lab17')
   for sub in ['docs', 'data', 'src']:
       folder = base / sub
       folder.mkdir(exist_ok=True)
       readme = folder / 'README.md'
       readme.write_text('# Пустой каталог\n')
   ```
   - Проверяйте наличие файлов с помощью `Path.is_file()`.

3. **Обработка исключений**  
   - Попытайтесь создать файл `src/README.md` повторно с режимом `x`.  
   - Обработайте `FileExistsError` и выведите сообщение.  
   ```python
   try:
       (base / 'src' / 'README.md').open('x')
   except FileExistsError:
       print('Файл уже существует')
   ```

4. **Проверка прав доступа**  
   - Попробуйте открыть файл `data/secret.txt` в режиме записи, если он не существует.  
   - Обработайте `PermissionError`, если текущий пользователь не имеет прав.  
   ```python
   try:
       (base / 'data' / 'secret.txt').open('w')
   except PermissionError:
       print('Нет прав на запись')
   ```

5. **Экспериментальная часть**  
   - Запустите программу несколько раз, меняя режимы открытия файлов (w, x, a).  
   - Сравните поведение при каждом режиме.  
   - Запишите наблюдения в таблицу (таблица должна быть включена в отчёт).
