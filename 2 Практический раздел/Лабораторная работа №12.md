# Лабораторная работа № 12  
Разработка программ обработки символов  

## Цель и задачи  
1. Научиться работать со строками в Python, использовать методы и функции для их обработки.  
2. Разработать программы, выполняющие поиск, замену, удаление и подсчёт символов в тексте.  
3. Оценить эффективность алгоритмов обработки символов и научиться анализировать результаты.  

## Теоретическая часть  
1. **Строка** – последовательность символов, неизменяемый объект.  
2. **Срез** – операция `s[start:stop:step]`, возвращает подстроку.  
3. **Метод `str.replace(old, new, count)`** – заменяет подстроку `old` на `new`.  
4. **Метод `str.count(sub)`** – возвращает количество неперекрывающихся вхождений `sub`.  
5. **Метод `str.find(sub, start, end)`** – индекс первого вхождения `sub`.  
6. **Метод `str.isdigit()`** – проверка, состоит ли строка только из цифр.  
7. **Алгоритм удаления пробелов** – `''.join(s.split())`.  
8. **Регулярные выражения** – модуль `re` позволяет искать и заменять сложные шаблоны.  
9. **Словарь частот** – `collections.Counter` для подсчёта символов.  
10. **Кодировка** – UTF‑8, важно при работе с кириллицей и спецсимволами.  

## Ход выполнения работы  
1. **Подготовка окружения**  
   ```python
   # Создаём рабочую папку и открываем файл
   import os
   os.makedirs('lab12', exist_ok=True)
   ```
2. **Чтение исходного текста**  
   ```python
   with open('lab12/input.txt', 'r', encoding='utf-8') as f:
       text = f.read()
   ```
3. **Удаление всех пробелов и переводов строки**  
   ```python
   cleaned = ''.join(text.split())
   print(cleaned[:100])  # первые 100 символов
   ```
4. **Подсчёт количества каждого символа**  
   ```python
   from collections import Counter
   freq = Counter(cleaned)
   for sym, cnt in freq.most_common(5):
       print(f'{sym}: {cnt}')
   ```
5. **Замена всех вхождений подстроки “Python” на “Py”**  
   ```python
   replaced = cleaned.replace('Python', 'Py')
   ```
6. **Поиск позиции первого вхождения символа ‘e’**  
   ```python
   pos = replaced.find('e')
   print(f'Первая позиция e: {pos}')
   ```
7. **Сохранение результатов в файл**  
   ```python
   with open('lab12/output.txt', 'w', encoding='utf-8') as f:
       f.write(f'Частоты: {freq}\n')
       f.write(f'Заменённый текст: {replaced[:200]}\n')
   ```
8. **Анализ эффективности**  
   - Время выполнения измеряемо через `time`  
   - Память – через `sys.getsizeof`
