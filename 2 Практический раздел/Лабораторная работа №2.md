# Лабораторная работа № 2 Создание линейных программ

## Цель и задачи
1. Научиться формулировать задачи линейного программирования (ЛП) и представлять их в стандартной форме.  
2. Овладеть базовыми инструментами Python для решения ЛП (библиотека `PuLP`), включая построение модели, добавление ограничений и целевой функции.  
3. Выполнить экспериментальное исследование влияния различных коэффициентов на решение задачи.  
4. Научиться анализировать и интерпретировать результаты, а также формулировать выводы в отчёте.

## Теоретическая часть
| Понятие | Определение |
|---------|-------------|
| **Линейная программа** | Уравнение вида \(\max/\min c^{T}x\) при линейных ограничениях \(Ax \le b\). |
| **Переменная решения** | Невырожденная переменная \(x_i \in \mathbb{R}\), обычно \(x_i \ge 0\). |
| **Целевая функция** | Функция, которую нужно максимизировать или минимизировать. |
| **Ограничения** | Линейные неравенства/равенства, ограничивающие допустимый набор решений. |
| **Проблема двойственности** | Для каждой ЛП существует двойственная задача с обратной целью. |
| **Формула симплекс-метода** | Итерационный алгоритм, переходящий от одного базисного решения к другому. |
| **Базисное решение** | Решение, получаемое при выборе базиса из переменных, равных нулю. |
| **Потенциально допустимое решение** | Решение, удовлетворяющее всем ограничениям, но не обязательно оптимальному. |
| **Система ограничений в канонической форме** | \(\sum a_{ij}x_j \le b_i, \; x_j \ge 0\). |
| **Кодирование в Python (PuLP)** | Библиотека `PuLP` реализует интерфейс для построения и решения ЛП. |

## Ход выполнения работы
1. **Установка библиотеки**  
   ```bash
   pip install pulp
   ```  
   Пояснение: `PuLP` – свободная библиотека Python для линейного программирования.

2. **Импортирование модулей**  
   ```python
   import pulp
   ```  
   Пример: Создание объекта задачи.

3. **Определение переменных**  
   ```python
   x1 = pulp.LpVariable('x1', lowBound=0)
   x2 = pulp.LpVariable('x2', lowBound=0)
   ```  
   Пояснение: переменные `x1` и `x2` неотрицательны.

4. **Создание задачи**  
   ```python
   prob = pulp.LpProblem('LP_Task', pulp.LpMaximize)
   ```  
   Пример: задаём тип задачи – максимизация.

5. **Добавление целевой функции**  
   ```python
   prob += 3*x1 + 5*x2, 'Objective'
   ```  
   Пояснение: цель – максимизировать \(3x_1 + 5x_2\).

6. **Добавление ограничений**  
   ```python
   prob += 2*x1 + x2 <= 20, 'Constraint_1'
   prob += x1 + 3*x2 <= 30, 'Constraint_2'
   prob += x1 >= 2, 'Constraint_3'
   ```  
   Пояснение: три ограничения, каждое с именем.

7. **Решение задачи**  
   ```python
   prob.solve()
   print('Status:', pulp.LpStatus[prob.status])
   for v in prob.variables():
       print(f'{v.name} = {v.varValue}')
   print('Optimal value:', pulp.value(prob.objective))
   ```  
   Пояснение: выводим статус, значения переменных и оптимальное значение.

8. **Эксперимент**  
   - Изменяем коэффициенты целевой функции (например, \(3 \rightarrow 4\)).  
   - Повторяем шаги 5–7.  
   - Сравниваем результаты.  

9. **Анализ результатов**  
   - Оцениваем влияние коэффициентов на оптимальное решение.  
   - Проверяем, какие ограничения стали активными (равенство).
