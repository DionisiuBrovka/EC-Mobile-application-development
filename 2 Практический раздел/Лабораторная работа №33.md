# Лабораторная работа № 33 Создание курсоров

## Цель и задачи
1. Освоить механизм работы курсоров в библиотеке `sqlite3` для Python.  
2. Научиться создавать, использовать и закрывать курсоры при выполнении запросов.  
3. Разобрать типы курсоров (`cursor`, `namedtuplecursor`, `row_factory`) и их применение.  
4. Оценить влияние использования курсоров на производительность и память.

## Теоретическая часть
| Понятие | Определение |
|--------|-------------|
| Курсор | Объект, обеспечивающий последовательный доступ к результатам SQL‑запроса. |
| `cursor()` | Метод объекта соединения, создающий новый курсор. |
| `execute(sql, parameters)` | Выполняет один SQL‑запрос с параметрами. |
| `fetchone()` | Возвращает одну строку результата. |
| `fetchmany(size)` | Возвращает `size` строк. |
| `fetchall()` | Возвращает все строки результата. |
| `row_factory` | Атрибут соединения, определяющий формат возвращаемых строк. |
| `namedtuplecursor` | Курсор, возвращающий строки как `namedtuple`. |
| Пакет `sqlite3` | Встроенная библиотека Python для работы с SQLite. |
| Параметризованный запрос | Запрос, где параметры передаются отдельно от SQL‑строки для предотвращения SQL‑инъекций. |
| База данных в памяти | Создаётся как `sqlite3.connect(':memory:')`. |

## Ход выполнения работы
1. **Подключение к базе данных**  
   ```python
   import sqlite3
   conn = sqlite3.connect(':memory:')   # создаём БД в памяти
   cursor = conn.cursor()
   ```
   *Пояснение:* `:memory:` создаёт временную БД, не сохраняющуюся на диске.

2. **Создание таблицы**  
   ```python
   cursor.execute('''
       CREATE TABLE students (
           id INTEGER PRIMARY KEY,
           name TEXT NOT NULL,
           age INTEGER
       )
   ''')
   ```
   *Пояснение:* таблица `students` будет хранить данные о студентах.

3. **Вставка данных через параметризованный запрос**  
   ```python
   students = [
       ('Alice', 21),
       ('Bob', 22),
       ('Charlie', 20)
   ]
   cursor.executemany('INSERT INTO students (name, age) VALUES (?, ?)', students)
   conn.commit()
   ```
   *Пояснение:* `executemany` ускоряет вставку множественных строк.

4. **Получение данных с помощью курсора**  
   ```python
   cursor.execute('SELECT * FROM students')
   for row in cursor.fetchall():
       print(row)
   ```
   *Пояснение:* `fetchall()` возвращает список кортежей, каждый элемент – строка таблицы.

5. **Использование `namedtuplecursor`**  
   ```python
   from collections import namedtuple
   conn.row_factory = lambda cursor, row: namedtuple('Student', [col[0] for col in cursor.description])(*row)
   cursor = conn.cursor()
   cursor.execute('SELECT * FROM students')
   for student in cursor.fetchall():
       print(student.name, student.age)
   ```
   *Пояснение:* каждая строка становится объектом `namedtuple`, что облегчает доступ к полям по имени.

6. **Закрытие курсора и соединения**  
   ```python
   cursor.close()
   conn.close()
   ```
   *Пояснение:* освобождаем ресурсы после завершения работы.
