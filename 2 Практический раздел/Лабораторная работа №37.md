# Лабораторная работа № 37  
Создание приложений с анимацией

## Цель и задачи

1. Ознакомиться с принципами работы графического интерфейса и анимации в Python.  
2. Научиться использовать библиотеку `tkinter` для создания простых анимированных приложений.  
3. Разработать собственное приложение с анимацией, демонстрируя навыки работы с графикой, таймерами и событиями.  
4. Научиться анализировать и описывать поведение программы в отчёте.

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| **Графический пользовательский интерфейс (GUI)** | Набор визуальных элементов (окна, кнопки, поля ввода), через которые пользователь взаимодействует с программой. |
| **Класс `Tk`** | Базовый объект окна в `tkinter`, создающий корневой контейнер для всех виджетов. |
| **Виджет** | Любой элемент интерфейса: `Button`, `Canvas`, `Label` и т.д. |
| **Канвас (`Canvas`)** | Специализированный виджет для рисования графики и анимации. |
| **Таймер (`after`)** | Метод, позволяющий вызвать функцию через заданный интервал времени (мс). |
| **Событие (`event`)** | Внешнее действие (нажатие клавиши, движение мыши), обрабатываемое программой. |
| **Состояние** | Текущее положение/значение переменных, влияющих на отображение интерфейса. |
| **Главный цикл (`mainloop`)** | Бесконечный цикл обработки событий, удерживающий окно открытым. |
| **Координаты** | Позиция объекта в пространстве канваса, измеряемая в пикселях. |
| **Библиотека `tkinter`** | Стандартная библиотека Python для создания GUI, включающая все вышеперечисленные элементы. |

## Ход выполнения работы

1. **Создание проекта и инициализация окна**  
   - Откройте новый файл `animation_app.py`.  
   - Импортируйте `tkinter` и создайте объект `root`.  
   ```python
   import tkinter as tk

   root = tk.Tk()
   root.title("Анимация на Tkinter")
   root.geometry("400x300")
   ```

2. **Добавление канваса**  
   - Создайте `Canvas`, поместите его в окно и задайте размеры.  
   ```python
   canvas = tk.Canvas(root, width=400, height=300, bg="white")
   canvas.pack()
   ```

3. **Рисование анимированного объекта**  
   - Нарисуйте прямоугольник (или круг) и сохраните его ID.  
   ```python
   rect = canvas.create_rectangle(10, 125, 60, 175, fill="blue")
   ```

4. **Определение функции перемещения**  
   - Создайте функцию, которая будет сдвигать объект и вызывать саму себя через `after`.  
   ```python
   def move():
       canvas.move(rect, 5, 0)  # смещение по X на 5 пикселей
       x1, _, x2, _ = canvas.coords(rect)
       if x2 < 400:
           root.after(50, move)  # вызов через 50 мс
   ```

5. **Запуск анимации**  
   - Вызовите `move()` после инициализации и запустите главный цикл.  
   ```python
   move()
   root.mainloop()
   ```

6. **Добавление пользовательских команд**  
   - Добавьте кнопку «Пауза/Продолжить» и обработчик событий, чтобы остановить/запустить анимацию.  
   ```python
   paused = False

   def toggle():
       nonlocal paused
       paused = not paused
       if not paused:
           move()

   pause_btn = tk.Button(root, text="Пауза", command=toggle)
   pause_btn.pack(pady=10)
   ```

7. **Тестирование**  
   - Запустите программу, проверьте, что прямоугольник движется, можно ставить паузу и возобновлять.  
   - Попробуйте изменить скорость, направление и форму объекта.
