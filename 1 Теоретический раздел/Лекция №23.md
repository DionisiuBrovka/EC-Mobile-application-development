## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Анимация** | Последовательность изображений, создающих иллюзию движения. |
| **Кадр** | Одиночный снимок в последовательности анимации. |
| **Приемник широковещательных намерений** | Компонент, который принимает и обрабатывает сообщения (intents) из глобальной сети приложений. |
| **Intent** | Объект, содержащий действие, параметры и данные для передачи между компонентами. |
| **BroadcastReceiver** | Класс, реализующий приемник для широковещательных сообщений. |
| **Matplotlib.animation.FuncAnimation** | Класс, позволяющий генерировать анимацию в matplotlib. |
| **Pygame.sprite.Group** | Коллекция спрайтов, упрощающая управление анимацией в Pygame. |
| **Frame rate (FPS)** | Количество кадров в секунду, определяющее плавность анимации. |
| **Delta time** | Временной интервал между кадрами, используемый для корректной анимации. |
| **Event loop** | Цикл, обрабатывающий пользовательские и системные события в приложении. |

## Примеры и иллюстрации

### Пример 1: Анимация с Matplotlib
```python
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

fig, ax = plt.subplots()
x = np.linspace(0, 2*np.pi, 100)
line, = ax.plot(x, np.sin(x))

def update(frame):
    line.set_ydata(np.sin(x + frame/10))
    return line,

ani = FuncAnimation(fig, update, frames=200, interval=50)
plt.show()
```

### Пример 2: Анимация в Pygame с BroadcastReceiver
```python
import pygame
import threading
import queue

# Инициализация Pygame
pygame.init()
screen = pygame.display.set_mode((400, 300))
clock = pygame.time.Clock()

# Событие для широковещания
UPDATE_EVENT = pygame.USEREVENT + 1
msg_queue = queue.Queue()

class BroadcastReceiver(threading.Thread):
    def run(self):
        while True:
            msg = msg_queue.get()
            if msg == "SPEED_UP":
                pygame.event.post(pygame.event.Event(UPDATE_EVENT, speed=2))
            elif msg == "SPEED_DOWN":
                pygame.event.post(pygame.event.Event(UPDATE_EVENT, speed=0.5))

receiver = BroadcastReceiver()
receiver.daemon = True
receiver.start()

# Графический объект
x_pos = 0
speed = 1

running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
        elif event.type == UPDATE_EVENT:
            speed = event.speed

    screen.fill((255, 255, 255))
    pygame.draw.rect(screen, (0, 100, 200), (x_pos, 100, 50, 50))
    x_pos += speed
    if x_pos > 400:
        x_pos = -50
    pygame.display.flip()
    clock.tick(60)

pygame.quit()
```
