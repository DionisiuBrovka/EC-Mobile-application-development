2. Научиться создавать и управлять уведомлениями в Android.  
3. Изучить основные настройки приложения в Android Studio.  
4. Подготовиться к ОКР №2 по модулю «Инструментальное программное обеспечение».

## Ключевые понятия
- **Служба (Service)** – компонент, работающий в фоновом режиме.  
- **Разрешение (Permission)** – доступ к ресурсам, защищённый системой.  
- **Уведомление (Notification)** – сообщение, отображаемое в статус‑баре.  
- **Broadcast Receiver** – компонент, реагирующий на системные события.  
- **Intent** – объект, описывающий действие, которое приложение должно выполнить.  
- **Notification Channel** – канал уведомлений, определяющий уровень важности.  
- **NotificationManager** – сервис, управляющий уведомлениями.  
- **Application Manifest** – файл, описывающий метаданные приложения.  
- **OKR (Objective and Key Results)** – методика постановки целей.  
- **Gradle** – система сборки Android‑приложений.

### 2. Уведомления и Notification Channel (30 мин)
#### 2.1. Структура уведомления  
- Заголовок, текст, иконка, интент.  

#### 2.2. Создание `NotificationChannel`  
```kotlin
val channel = NotificationChannel(
    "default",
    "Default Channel",
    NotificationManager.IMPORTANCE_DEFAULT
).apply {
    description = "Channel for general notifications"
}
notificationManager.createNotificationChannel(channel)
```

#### 2.3. Отправка уведомления  
```kotlin
val notification = NotificationCompat.Builder(this, "default")
    .setContentTitle("Задача завершена")
    .setContentText("Нажмите, чтобы открыть")
    .setSmallIcon(R.drawable.ic_done)
    .setContentIntent(pendingIntent)
    .build()
notificationManager.notify(1001, notification)
```
