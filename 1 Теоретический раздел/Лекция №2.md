# Лексика языка. Типы данных. Приведение типов. Операторы

## Цели и задачи лекции
- Понять структуру и роль лексики в Python.
- Ознакомиться с основными типами данных и их характеристиками.
- Научиться корректно приводить типы и избегать ошибок.
- Освоить основные арифметические, логические и сравнения операторы.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Лексема | Самый маленький элемент языка, имеющий смысл (ключевое слово, идентификатор, литерал). |
| Тип данных | Класс, описывающий свойства и поведение переменной. |
| Нумерический тип | Типы, представляющие числа: `int`, `float`, `complex`. |

| Логический тип | Тип `bool` с двумя значениями `True` и `False`. |
| Приведение типов | Явное преобразование значения из одного типа в другой. |
| Оператор | Символ, выполняющий действие над операндами. |
| Арифметический оператор | Оператор, выполняющий математические операции. |
| Логический оператор | Оператор, сравнивающий значения и возвращающий логический результат. |
| Оператор сравнения | Оператор, проверяющий отношение между операндами. |

## Основное содержание

### 1. Лексика Python (15 мин)
- **Идентификаторы** – имена переменных, функций, классов. Правила: `[a-zA-Z_][a-zA-Z0-9_]*`.
- **Ключевые слова** – зарезервированные слова (`if`, `for`, `class` и др.).
- **Литералы** – фиксированные значения: числа, строки, булевы.
- **Разделители** – скобки, запятые, точки с запятой.
- **Комментарии** – строки, начинающиеся с `#`.

```python
# Комментарий
x = 42          # целое число
y = 3.14        # дробное число
name = "Alice"  # строка
flag = True     # логическое
```

### 2. Типы данных (25 мин)
#### 2.1 Нумерические типы
- `int`: целые числа без ограничений по размеру.
- `float`: числа с плавающей точкой двойной точности.
- `complex`: комплексные числа `a + bj`.

```python
a = 5          # int
b = 2.7        # float
c = 1 + 2j     # complex
```

#### 2.2 Строковый тип
- Последовательность символов, неизменяемая.
- Операции: конкатенация (`+`), повторение (`*`), индексация, срезы.

```python
s = "Hello, world!"
print(s[7:12])  # world
print(s * 3)    # Hello, world!Hello, world!Hello, world!
```

#### 2.3 Логический тип
- `True`, `False`.
- Используется в условных выражениях.

```python
is_active = False
if is_active:
    print("Active")
```

#### 2.4 Списки, кортежи, множества и словари (5 мин)
- Списки (`list`) – изменяемые, упорядоченные.
- Кортежи (`tuple`) – неизменяемые, упорядоченные.
- Множества (`set`) – неупорядоченные, уникальные элементы.
- Словари (`dict`) – пары ключ‑значение.

### 3. Приведение типов (20 мин)
- **Явное приведение**: `int()`, `float()`, `str()`, `bool()`.
- **Неявное приведение** в выражениях (`int` + `float` → `float`).
- Ошибки при несовместном преобразовании.

```python
x = "10"
y = int(x)          # 10
z = float(x) + 2    # 12.0

# Ошибка
# a = "abc"
# b = int(a)  # ValueError
```

### 4. Операторы (20 мин)
#### 4.1 Арифметические операторы
| Оператор | Описание | Пример |
|----------|----------|--------|
| `+` | сложение | `a + b` |
| `-` | вычитание | `a - b` |
| `*` | умножение | `a * b` |
| `/` | деление (результат float) | `a / b` |
| `//` | целочисленное деление | `a // b` |
| `%` | остаток | `a % b` |
| `**` | возведение в степень | `a ** b` |

#### 4.2 Логические операторы
- `and`, `or`, `not`.

```python
a = True
b = False
print(a and b)  # False
print(not a)    # False
```

#### 4.3 Операторы сравнения
| Оператор | Описание | Пример |
|----------|----------|--------|
| `==` | равно | `a == b` |
| `!=` | не равно | `a != b` |
| `>` | больше | `a > b` |
| `<` | меньше | `a < b` |
| `>=` | больше или равно | `a >= b` |
| `<=` | меньше или равно | `a <= b` |

#### 4.4 Операторы присваивания
- `=`, `+=`, `-=`, `*=`, `/=`, `%=` и т.д.

```python
x = 5
x += 3   # x = 8
```
