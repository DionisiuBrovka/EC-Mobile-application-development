# Реализация основных и дополнительных функциональностей

## Цели и задачи лекции
- Понять, какие основные и дополнительные функциональности можно реализовать в Python‑приложениях.  
- Освоить принципы проектирования и реализации модулей, классов, функций.  
- Научиться использовать стандартные библиотеки и сторонние пакеты для расширения возможностей программы.  
- Попрактиковаться в создании простых скриптов с демонстрацией работы функциональных блоков.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Модуль** | Файл с расширением `.py`, содержащий функции, классы и переменные. |
| **Функция** | Набор инструкций, которые выполняются при вызове. |
| **Класс** | Шаблон для создания объектов с атрибутами и методами. |
| **Библиотека** | Коллекция готовых модулей, предоставляющих функциональность. |
| **API** | Интерфейс программирования приложений – набор функций, классов и протоколов. |
| **Пакет** | Папка с модулями и файлом `__init__.py`. |
| **Декоратор** | Функция, которая принимает другую функцию и возвращает модифицированную версию. |
| **Исключение** | Событие, возникающее при ошибке, которое можно обработать. |
| **Микросервис** | Независимый сервис, реализующий конкретную функциональность. |
| **Контейнер** | Изолированная среда, в которой запускается приложение. |

## Основное содержание

### 1. Организация кода (30 мин)
#### 1.1. Структура проекта (10 мин)
- Папка `src/` с модулями.  
- Папка `tests/` с тестами.  
- Файл `setup.py` или `pyproject.toml` для управления зависимостями.

#### 1.2. Модули и пакеты (10 мин)
```python
# src/utils/helpers.py
def greet(name: str) -> str:
    return f"Привет, {name}!"
```

#### 1.3. Импорт и namespace (10 мин)
```python
from utils.helpers import greet
print(greet("Алексей"))
```

### 2. Основные функциональности (35 мин)
#### 2.1. Работа с данными (10 мин)
- Чтение/запись CSV, JSON, XML.  
```python
import json

data = {"name": "Иван", "age": 30}
with open('data.json', 'w') as f:
    json.dump(data, f)
```

#### 2.2. Операции с файлами (10 мин)
- Чтение/запись текстовых и бинарных файлов.  
```python
with open('log.txt', 'a') as f:
    f.write('Новая запись\n')
```

#### 2.3. Обработка ошибок (10 мин)
```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Ошибка: {e}")
```

#### 2.4. Логирование (5 мин)
```python
import logging
logging.basicConfig(level=logging.INFO)
logging.info("Приложение запущено")
```

### 3. Дополнительные функциональности (25 мин)
#### 3.1. Асинхронность (10 мин)
```python
import asyncio

async def fetch(url):
    return await asyncio.sleep(1, result=f"Fetched {url}")

asyncio.run(fetch("http://example.com"))
```

#### 3.2. Работа с базами данных (10 мин)
- Использование `sqlite3` или `SQLAlchemy`.  
```python
import sqlite3
conn = sqlite3.connect('example.db')
cursor = conn.cursor()
cursor.execute('CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)')
conn.commit()
```

#### 3.3. Создание CLI (5 мин)
- Библиотека `argparse`.  
```python
import argparse
parser = argparse.ArgumentParser(description='Пример CLI')
parser.add_argument('--name', help='Ваше имя')
args = parser.parse_args()
print(f"Привет, {args.name}!")
```

## Примеры и иллюстрации
```python
# Полный пример: модуль, класс, логирование, исключения

# src/app.py
import logging

logging.basicConfig(level=logging.INFO)

class Calculator:
    def add(self, a, b):
        return a + b

    def divide(self, a, b):
        if b == 0:
            raise ValueError("Деление на ноль")
        return a / b

def main():
    calc = Calculator()
    print("Сложение: ", calc.add(5, 7))
    try:
        print("Деление: ", calc.divide(5, 0))
    except ValueError as e:
        logging.error(e)

if __name__ == "__main__":
    main()
```
