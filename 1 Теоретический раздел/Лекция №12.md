# Система ввода-вывода. Подготовка к ОКР №1

- Научиться работать с потоками данных: чтение/запись файлов, стандартного ввода/вывода.  
- Понять принципы обработки ошибок при работе с I/O.  
- Подготовиться к практическому заданию ОКР №1.

## Ключевые понятия
- **Ввод‑вывод (I/O)** – обмен данными между программой и внешними устройствами.  
- **Поток (stream)** – последовательность байт, передаваемая по каналу.  
- **Файл** – логически организованный поток данных на диске.  
- **Модуль `open()`** – функция для открытия файлов, возвращающая объект‑поток.  
- **Контекстный менеджер (`with`)** – конструкция, обеспечивающая корректное закрытие ресурсов.  
- **Кодировка** – способ представления текста в байтах.  
- **Буферизация** – временное хранение данных в памяти для ускорения I/O.  
- **Исключения (`IOError`, `OSError`)** – ошибки, возникающие при работе с I/O.

### 2. Работа с файлами (25 мин)
#### 2.1 Открытие и закрытие файлов (5 мин)
```python
file = open('example.txt', 'r', encoding='utf-8')
# ...
file.close()
```
#### 2.2 Чтение данных (10 мин)
- `read()`, `readline()`, `readlines()`.  
```python
with open('example.txt', 'r', encoding='utf-8') as f:
    content = f.read()
```
#### 2.3 Запись данных (10 мин)
- `write()`, `writelines()`.  
```python
with open('output.txt', 'w', encoding='utf-8') as f:
    f.write('Hello, world!\n')
```

### 3. Стандартный ввод/вывод (15 мин)
#### 3.1 `print()` и `input()` (5 мин)
#### 3.2 Перенаправление потоков (10 мин)
```python
import sys
sys.stdout.write('Direct write to stdout\n')
```

### 4. Обработка ошибок I/O (10 мин)
- Исключения `IOError`, `OSError`.  
- Использование блока `try/except`.  
```python
try:
    with open('nonexistent.txt', 'r') as f:
        data = f.read()
except FileNotFoundError:
    print('File not found')
```

### 5. Буферизация и оптимизация (10 мин)
- Параметр `buffering` в `open()`.  
- Пакетная обработка больших файлов.

## Примеры и иллюстрации

### Пример 1: Чтение CSV файла
```python
import csv

with open('data.csv', 'r', encoding='utf-8') as csvfile:
    reader = csv.reader(csvfile)
    for row in reader:
        print(row)
```

### Пример 2: Запись JSON
```python
import json

data = {'name': 'Alice', 'age': 30}
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, indent=4)
```

### Пример 3: Асинхронное чтение (Python 3.7+)
```python
import asyncio

async def read_file():
    loop = asyncio.get_running_loop()
    with open('bigfile.txt', 'r', encoding='utf-8') as f:
        content = await loop.run_in_executor(None, f.read)
    return content
```
