# Работа с текстом

## Цели и задачи лекции
- Изучить основные типы текстовых данных в Python и их представление.
- Освоить методы обработки строк: поиск, замену, разбиение и объединение.
- Познакомиться с регулярными выражениями и их применением.
- Практически применить методы работы с текстом для решения типичных задач.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Строка (string) | Последовательность символов, хранящаяся в переменной. |
| Символ | Самый маленький элемент строки, может быть буквой, цифрой, пробелом. |
| Метка (label) | Отметка в тексте, используемая для поиска. |
| Регулярное выражение (regex) | Спецификация шаблона, описывающего набор строк. |
| Методы строк | Встроенные функции, применяемые к объектам типа str. |
| Форматирование строк | Создание строк с переменными через f-строки, `%` и `.format()`. |
| Unicode | Стандарт кодировки символов, позволяющий работать с многоязычными текстами. |
| Текстовый поток | Поток данных, читаемый и записываемый через файлы. |
| Файловый объект | Объект, представляющий открытый файл. |
| Пакет `re` | Стандартная библиотека Python для работы с регулярными выражениями. |

## Основное содержание
### 1. Представление текста в Python (30 мин)
- Создание строк: одинарные, двойные, тройные кавычки.
- Экранирование символов (`\\`, `\'`, `\"`, `\n`, `\t`).
- Функции `len()`, `isalpha()`, `isdigit()`, `isspace()`.

```python
s = "Hello, world!\nПривет, мир!"
print(len(s))          # длина строки
print(s.isalpha())     # False, содержит пробелы и пунктуацию
print(s[7:12])         # срез строки
```

### 2. Методы строк (25 мин)
- Изменение регистра: `upper()`, `lower()`, `title()`.
- Поиск и замена: `find()`, `index()`, `replace()`.
- Разделение и объединение: `split()`, `join()`.
- Проверка начала/конца: `startswith()`, `endswith()`.
- Форматирование: f-строки, `%` и `.format()`.

```python
email = "user@example.com"
domain = email.split('@')[1]
print(domain)  # example.com
```

### 3. Регулярные выражения (25 мин)
- Конструкция модуля `re`: `match`, `search`, `findall`, `sub`.
- Метасимволы: `.` `*` `+` `?` `[]` `^` `$`.
- Группировка и квантификаторы.
- Примеры: проверка почтового адреса, извлечение дат.

```python
import re

pattern = r'^\w+@\w+\.\w{2,3}$'
print(re.match(pattern, 'test@mail.com'))  # <re.Match object>
```

### 4. Работа с файлами (10 мин)
- Открытие и закрытие файлов: `open()`, контекстный менеджер `with`.
- Чтение строк: `read()`, `readline()`, `readlines()`.
- Запись в файл: `write()`, `writelines()`.
- Кодировка: `encoding='utf-8'`.

```python
with open('data.txt', 'r', encoding='utf-8') as f:
    for line in f:
        print(line.strip())
```

## Примеры и иллюстрации
1. **Разбиение текста на слова и подсчет частоты**  
   ```python
   from collections import Counter
   text = "one two two three three three"
   words = text.split()
   freq = Counter(words)
   print(freq)  # Counter({'three': 3, 'two': 2, 'one': 1})
   ```

2. **Извлечение телефонного номера**  
   ```python
   phone_pattern = r'\+?\d{1,3}[-.\s]?\(?\d{3}\)?[-.\s]?\d{3}[-.\s]?\d{4}'
   text = "Call me at +1 (555) 123-4567 tomorrow."
   match = re.search(phone_pattern, text)
   if match:
       print(match.group())  # +1 (555) 123-4567
   ```

3. **Форматирование даты**  
   ```python
   from datetime import datetime
   raw = "2025-11-20"
   dt = datetime.strptime(raw, "%Y-%m-%d")
   print(dt.strftime("%d.%m.%Y"))  # 20.11.2025
   ```
