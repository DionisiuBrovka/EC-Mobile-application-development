# Виджеты-списки и привязка данных. Меню. Диалоговые окна

## Цели и задачи лекции
- Познакомить студентов с основными видами виджетов, используемых для отображения и управления списками данных в графических приложениях.
- Рассказать о механизме привязки данных к виджетам и его преимуществах.
- Осветить принципы работы меню и диалоговых окон, их роль в пользовательском интерфейсе.
- Предоставить практические примеры реализации описанных компонентов на Python с использованием библиотеки Tkinter.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Виджет | Самостоятельный элемент графического интерфейса, отвечающий за ввод/вывод данных. |
| Привязка данных | Механизм автоматической синхронизации состояния виджета с переменной или моделью данных. |
| Меню | Компонент интерфейса, предоставляющий набор пунктов для выполнения команд. |
| Диалоговое окно | Окно, которое требует взаимодействия пользователя перед продолжением работы программы. |
| Listbox | Виджет, отображающий список строк, из которых пользователь может выбирать элементы. |
| Combobox | Виджет, объединяющий поле ввода и выпадающий список. |
| Entry | Виджет однострочного текста, позволяющий пользователю вводить данные. |
| Text | Виджет многострочного текста, поддерживающий форматирование и редактирование. |
| Button | Виджет, реагирующий на нажатие и выполняющий заданную функцию. |
| Variable | Специальный объект Tkinter (`StringVar`, `IntVar` и др.), который хранит данные и позволяет привязать их к виджетам. |

## Примеры и иллюстрации

```python
import tkinter as tk
from tkinter import ttk, messagebox, filedialog, simpledialog

root = tk.Tk()
root.title("Пример виджетов")

# Listbox
listbox = tk.Listbox(root, height=5)
listbox.pack(pady=5)
for item in ["Python", "Java", "C++", "JavaScript"]:
    listbox.insert(tk.END, item)

# Combobox
combo = ttk.Combobox(root, values=["Apple", "Banana", "Cherry"])
combo.current(0)
combo.pack(pady=5)

# Привязка данных
name_var = tk.StringVar()
entry = tk.Entry(root, textvariable=name_var)
entry.pack(pady=5)

label = tk.Label(root, text="Ваше имя:")
label.pack(pady=5)

# Меню
menubar = tk.Menu(root)
file_menu = tk.Menu(menubar, tearoff=0)
file_menu.add_command(label="Открыть", command=lambda: messagebox.showinfo("Открыть", "Открыть файл"))
file_menu.add_command(label="Сохранить", command=lambda: messagebox.showinfo("Сохранить", "Сохранить файл"))
file_menu.add_separator()
file_menu.add_command(label="Выход", command=root.quit)
menubar.add_cascade(label="Файл", menu=file_menu)
root.config(menu=menubar)

# Диалоговое окно
def add_task():
    task = simpledialog.askstring("Задача", "Введите задачу:")
    if task:
        listbox.insert(tk.END, task)

btn = tk.Button(root, text="Добавить задачу", command=add_task)
btn.pack(pady=10)

root.mainloop()
```

- Виджеты-списки (Listbox, Combobox) позволяют удобно отображать и выбирать данные.

- Меню служит навигацией и организацией команд пользователя.
- Диалоговые окна — ключевой элемент взаимодействия, позволяющий получить ввод или показать информацию.
