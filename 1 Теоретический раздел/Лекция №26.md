# Разработка макетов и активностей приложения

## Цели и задачи лекции
- Понять роль макетов и активностей в структуре приложения.
- Освоить инструменты создания макетов и активностей в Python.
- Научиться проектировать простые пользовательские интерфейсы.
- Применить полученные знания на практических примерах.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Макет (layout)** | Структурное описание размещения элементов интерфейса. |
| **Активность (activity)** | Конкретный экран или модуль приложения с логикой взаимодействия. |
| **Компонент (widget)** | Базовый элемент пользовательского интерфейса (кнопка, поле ввода и т.д.). |
| **Событие (event)** | Действие пользователя, которое приложение обрабатывает. |
| **Контейнер (container)** | Элемент, внутри которого размещаются другие компоненты (панель, окно). |
| **Фреймворк** | Набор библиотек, упрощающий разработку GUI. |
| **Callback** | Функция, вызываемая в ответ на событие. |
| **Кодогенерация** | Автоматическое создание исходного кода макета из графического редактора. |
| **Стилизация (styling)** | Применение внешнего вида к компонентам через CSS‑подобные правила. |
| **Модульность** | Разделение приложения на независимые, переиспользуемые части. |

## Основное содержание

### 1. Архитектура GUI‑приложения (15 мин)
- Иерархия окон → активностей → компонентов.
- Принцип Model‑View‑Controller (MVC) в Python‑GUI.
- Паттерн «Разделение представления и логики».

### 2. Выбор фреймворка и инструменты (20 мин)
- Tkinter – стандартный, легкий, подходит для учебы.
- PyQt/PySide – более мощный, но требует лицензии.
- Kivy – кросс‑платформенный, для сенсорных интерфейсов.
- Интеграция с IDE (PyCharm, VS Code).

### 3. Создание макетов в Tkinter (25 мин)
- Создание главного окна `Tk()`.
- Использование контейнеров `Frame`, `LabelFrame`.
- Расположение компонентов: `pack`, `grid`, `place`.
- Пример простого макета:
```python
import tkinter as tk

root = tk.Tk()
root.title("Пример макета")

frame = tk.Frame(root, padx=10, pady=10)
frame.pack(fill="both", expand=True)

label = tk.Label(frame, text="Введите имя:")
label.grid(row=0, column=0, sticky="e")

entry = tk.Entry(frame)
entry.grid(row=0, column=1, sticky="we")

button = tk.Button(frame, text="Отправить")
button.grid(row=1, column=0, columnspan=2, pady=5)

root.mainloop()
```

### 4. Активности как классы (20 мин)
- Определение активности как подкласса `tk.Frame`.
- Инициализация, привязка событий.
- Переключение между активностями в одном окне.
```python
class MainActivity(tk.Frame):
    def __init__(self, master):
        super().__init__(master)
        tk.Label(self, text="Главная активность").pack()
        tk.Button(self, text="Перейти", command=self.open_second).pack()

    def open_second(self):
        self.pack_forget()
        second = SecondActivity(self.master)
        second.pack(fill="both", expand=True)

class SecondActivity(tk.Frame):
    def __init__(self, master):
        super().__init__(master)
        tk.Label(self, text="Вторая активность").pack()
        tk.Button(self, text="Назад", command=self.go_back).pack()

    def go_back(self):
        self.pack_forget()
        main = MainActivity(self.master)
        main.pack(fill="both", expand=True)
```

### 5. События и колбэки (15 мин)
- Типы событий: `Button-1`, `KeyPress`, `FocusIn`.
- Метод `bind` и обработчики.
- Пример: изменение цвета фона при наведении.
```python
def on_enter(e):
    e.widget.config(bg="lightblue")

def on_leave(e):
    e.widget.config(bg="SystemButtonFace")

btn = tk.Button(frame, text="Наведите мышью")
btn.grid(row=2, column=0, columnspan=2, pady=5)
btn.bind("<Enter>", on_enter)
btn.bind("<Leave>", on_leave)
```

### 6. Стилизация интерфейса (10 мин)
- Использование темы `ttk` и `style`.
- Создание пользовательских стилей.

```python
from tkinter import ttk

style = ttk.Style()
style.configure("TButton", font=("Arial", 12), foreground="white", background="#4CAF50")
```
