# Компонент RecyclerView и выдвижные панели

## Цели и задачи лекции
1. Познакомить студентов с архитектурой и назначением `RecyclerView`.  
2. Показать, как реализовать выдвижные панели (DrawerLayout) в приложении.  
3. Научить интегрировать `RecyclerView` в выдвижную панель и управлять данными.  
4. Предоставить практические навыки работы с адаптерами и ViewHolder.

## Ключевые понятия
- **RecyclerView** – компонент для отображения больших списков данных с возможностью повторного использования ячеек.  
- **Adapter** – класс, связывающий данные с `RecyclerView`.  
- **ViewHolder** – объект, содержащий ссылки на элементы разметки ячейки.  
- **LayoutManager** – отвечает за позиционирование элементов внутри `RecyclerView`.  
- **DrawerLayout** – контейнер, реализующий выдвижные панели (navigation drawer).  
- **NavigationView** – виджет, отображающий список пунктов в выдвижной панели.  
- **DiffUtil** – утилита для оптимизированного обновления списка.  
- **ItemDecoration** – класс, добавляющий отступы, разделители и т.п. к элементам `RecyclerView`.  
- **ViewBinding / DataBinding** – технологии привязки разметки к коду.  
- **LiveData / ViewModel** – архитектурные компоненты для управления UI‑данными.

## Основное содержание

### 1. Введение в RecyclerView (15 мин)
- История появления `ListView` → `RecyclerView`.  
- Почему `RecyclerView` более гибок: различные LayoutManager, анимации, оптимизация.  
- Структура проекта: `RecyclerView` в XML, `Adapter`, `ViewHolder`.

#### Пример разметки
```xml
<androidx.recyclerview.widget.RecyclerView
    android:id="@+id/recyclerView"
    android:layout_width="match_parent"
    android:layout_height="match_parent" />
```

### 2. Создание простого адаптера (20 мин)
- Определяем модель данных: `data class Item(val id: Int, val title: String)`.  
- Создаём `ItemViewHolder`.  
- Реализуем `Adapter` с `onCreateViewHolder`, `onBindViewHolder`, `getItemCount`.  
- Добавляем простое `LinearLayoutManager`.

#### Код адаптера
```kotlin
class ItemAdapter(private val items: List<Item>) :
    RecyclerView.Adapter<ItemAdapter.ItemViewHolder>() {

    inner class ItemViewHolder(view: View) : RecyclerView.ViewHolder(view) {
        val title: TextView = view.findViewById(R.id.itemTitle)
    }

    override fun onCreateViewHolder(parent: ViewGroup, viewType: Int): ItemViewHolder {
        val view = LayoutInflater.from(parent.context)
            .inflate(R.layout.item_view, parent, false)
        return ItemViewHolder(view)
    }

    override fun onBindViewHolder(holder: ItemViewHolder, position: Int) {
        holder.title.text = items[position].title
    }

    override fun getItemCount() = items.size
}
```

### 3. Выдвижная панель (DrawerLayout) (20 мин)
- Структура разметки: `DrawerLayout` → `NavigationView` + основной контент.  
- Настройка `ActionBarDrawerToggle`.  
- Обработка кликов по пунктам меню.

#### Разметка DrawerLayout
```xml
<androidx.drawerlayout.widget.DrawerLayout
    android:id="@+id/drawerLayout"
    android:layout_width="match_parent"
    android:layout_height="match_parent">

    <!-- Основной контент -->
    <FrameLayout
        android:id="@+id/contentFrame"
        android:layout_width="match_parent"
        android:layout_height="match_parent"/>

    <!-- Выдвижная панель -->
    <com.google.android.material.navigation.NavigationView
        android:id="@+id/navigationView"
        android:layout_width="wrap_content"
        android:layout_height="match_parent"
        android:layout_gravity="start"
        app:menu="@menu/drawer_menu"/>
</androidx.drawerlayout.widget.DrawerLayout>
```

### 4. Интеграция RecyclerView в DrawerLayout (15 мин)
- Добавляем `RecyclerView` в `NavigationView` через `app:headerLayout` или `app:menu`.  
- Обновляем данные списка при открытии/закрытии панели.  
- Используем `DiffUtil` для плавного обновления.

#### Пример обновления списка
```kotlin
val diffCallback = object : DiffUtil.ItemCallback<Item>() {
    override fun areItemsTheSame(oldItem: Item, newItem: Item) = oldItem.id == newItem.id
    override fun areContentsTheSame(oldItem: Item, newItem: Item) = oldItem == newItem
}
val adapter = ListAdapter(diffCallback)
recyclerView.adapter = adapter
adapter.submitList(newList)
```

### 5. Оптимизация и пользовательские эффекты (10 мин)
- Добавление разделителей через `ItemDecoration`.  
- Анимация при добавлении/удалении элементов.  
- Использование `ViewBinding` для упрощения кода.

#### ItemDecoration
```kotlin
class DividerDecoration : RecyclerView.ItemDecoration() {
    override fun getItemOffsets(outRect: Rect, view: View,
                                parent: RecyclerView, state: RecyclerView.State) {
        outRect.bottom = 1.dp // 1dp разделитель
    }
}
```

- Интеграция `RecyclerView` в выдвижную панель расширяет возможности UI и повышает интерактивность.  
- Использование `DiffUtil` и `ItemDecoration` повышает производительность и эстетику приложения.
