# Введение. Объектно‑ориентированное программирование и язык программирования Java

## Цели и задачи лекции
- Понять основные принципы объектно‑ориентированного программирования (ООП).  
- Ознакомиться с историей и особенностями языка Java.  
- Научиться писать простые программы на Java, используя классы и объекты.  
- Развить навыки логического мышления при работе с абстракциями.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Объект | Экземпляр класса, содержащий состояние и поведение. |
| Класс | Шаблон для создания объектов, описывающий их поля и методы. |
| Наследование | Механизм, позволяющий создавать новый класс на основе существующего. |
| Полиморфизм | Возможность использовать объекты разных классов через один интерфейс. |
| Инкапсуляция | Сокрытие внутренней реализации объекта от внешнего мира. |
| Абстракция | Выделение существенных свойств объекта, игнорирование деталей. |
| Интерфейс | Набор абстрактных методов, реализуемых классами. |
| Модификатор доступа | Ключевое слово (`public`, `private`, `protected`) определяющее видимость члена. |
| Конструктор | Специальный метод, создающий объект. |
| Пакет | Организационная единица, сгруппировывающая связанные классы. |

## Основное содержание

### 1. История и философия Java (15 мин)
- Создание Sun Microsystems, выпуск Java 1.0 в 1995 г.  
- Ключевые принципы: *Write Once, Run Anywhere* (WORA), безопасность, многопоточность.  
- Сравнение Java с Python: статическая типизация, JVM, сборка мусора.

### 2. Синтаксис и базовые конструкции (20 мин)
- Структура программы: `public class ClassName { public static void main(String[] args) { … } }`.  
- Переменные, типы, операторы, условные конструкции, циклы.  
- Пример: простая программа «Hello, World!».

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

### 3. Объекты и классы (20 мин)
- Определение класса: поля, методы, конструкторы.  
- Создание объекта: `ClassName obj = new ClassName();`.  
- Пример: класс `Person` с полями `name`, `age` и методом `greet()`.

```java
public class Person {
    private String name;
    private int age;

    public Person(String n, int a) {
        name = n;
        age = a;
    }

    public void greet() {
        System.out.println("Hi, I'm " + name + ", " + age + " years old.");
    }
}
```

### 4. Основы ООП в Java (20 мин)
- Инкапсуляция: `private` поля, `public` геттеры/сеттеры.  
- Наследование: `class Student extends Person`.  
- Полиморфизм: метод `greet()` переопределяется в подклассе.  
- Абстракция: абстрактный класс `Animal` и интерфейс `Flyable`.

```java
public abstract class Animal {
    public abstract void sound();
}

public class Dog extends Animal {
    @Override
    public void sound() {
        System.out.println("Bark");
    }
}
```

### 5. Работа с пакетами и модулями (15 мин)
- Синтаксис пакета: `package com.example;`.  
- Организация файловой структуры.  
- Модульная система Java 9+: `module-info.java`.

```java
module com.example.myapp {
    requires java.base;
}
```

## Примеры и иллюстрации
- Краткая программа «Калькулятор» с использованием классов.  
- Схема наследования: `Object → Animal → Dog`.  

```java
public class Calculator {
    public static double add(double a, double b) {
        return a + b;
    }
}
```
