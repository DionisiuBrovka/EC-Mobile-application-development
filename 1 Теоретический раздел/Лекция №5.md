# Классы и объекты

## Цели и задачи лекции  
- Понять, что такое классы и объекты в Python.  
- Освоить синтаксис объявления классов и создание экземпляров.  
- Изучить основные принципы объектно‑ориентированного программирования: инкапсуляция, наследование, полиморфизм.  
- Научиться использовать методы и свойства объектов в практических задачах.

## Ключевые понятия  
- **Класс** – шаблон, описывающий структуру и поведение объектов.  
- **Объект (экземпляр)** – конкретная реализация класса с собственными данными.  
- **Атрибут** – переменная, принадлежащая классу или объекту.  
- **Метод** – функция, объявленная внутри класса и вызываемая через объект.  
- **Конструктор (`__init__`)** – специальный метод, инициализирующий новый объект.  
- **Наследование** – механизм создания нового класса на основе существующего.  
- **Полиморфизм** – возможность использовать объекты разных классов через один интерфейс.  
- **Инкапсуляция** – скрытие внутренней реализации от внешнего кода.  
- **Магические методы** – методы, именованные как `__метод__`, реализующие специальные поведения.  
- **Классовый атрибут** – переменная, общая для всех экземпляров класса.

### 2. Создание классов и объектов (20 мин)  
- Синтаксис объявления класса.  
- Конструктор `__init__` и инициализация атрибутов.  
- Создание экземпляров и обращение к атрибутам.  

```python
class Car:
    def __init__(self, make, model, year):
        self.make = make          # атрибут экземпляра
        self.model = model
        self.year = year

my_car = Car("Toyota", "Corolla", 2020)
print(my_car.make)  # Toyota
```

### 3. Методы и их типы (15 мин)  
- Регулярные методы (`self`).  
- Классовые методы (`@classmethod`).  
- Статические методы (`@staticmethod`).  

```python
class Circle:
    pi = 3.14159  # классовый атрибут

    def __init__(self, radius):
        self.radius = radius

    def area(self):  # обычный метод
        return Circle.pi * self.radius ** 2

    @classmethod
    def unit_circle(cls):  # класс-метод
        return cls(1)

    @staticmethod
    def description():  # статический метод
        return "Круг – геометрическая фигура."
```

### 4. Инкапсуляция и доступ к атрибутам (15 мин)  
- Публичные, защищённые и приватные атрибуты.  
- Геттеры и сеттеры (принцип `@property`).  

```python
class BankAccount:
    def __init__(self, balance=0):
        self.__balance = balance  # приватный атрибут

    @property
    def balance(self):
        return self.__balance

    @balance.setter
    def balance(self, amount):
        if amount >= 0:
            self.__balance = amount
```

### 5. Наследование и полиморфизм (15 мин)  
- Создание подклассов, переопределение методов.  
- Использование `super()`.  

```python
class Animal:
    def speak(self):
        print("Some sound")

class Dog(Animal):
    def speak(self):          # переопределение
        print("Woof!")

class Cat(Animal):
    def speak(self):
        print("Meow!")

def make_sound(animal):
    animal.speak()  # полиморфизм

make_sound(Dog())
make_sound(Cat())
```

### 6. Практическое задание: проект «Библиотека» (15 мин)  
Разработать классы `Book`, `Author`, `Library` с взаимосвязями и методами для добавления/вывода информации.  

```python
class Author:
    def __init__(self, name):
        self.name = name

class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author

class Library:
    def __init__(self):
        self.books = []

    def add_book(self, book):
        self.books.append(book)

    def list_books(self):
        for b in self.books:
            print(f"{b.title} by {b.author.name}")
```

## Выводы по лекции  
Классы и объекты – фундаментальные конструкции Python, позволяющие организовывать код в логически связанные модули. Понимание методов, атрибутов, наследования и инкапсуляции открывает возможности для создания масштабируемых и поддерживаемых программ.
