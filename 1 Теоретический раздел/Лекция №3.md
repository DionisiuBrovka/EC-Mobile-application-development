# Управляющие инструкции

## Цели и задачи лекции
- Понять роль управляющих инструкций в процессе выполнения программы.
- Изучить синтаксис и семантику условных и циклических конструкций в Python.
- Научиться выбирать подходящий тип управляющей инструкции для решения задач.
- Применить полученные знания на практических примерах.

## Ключевые понятия
- **Управляющая инструкция** – команда, определяющая порядок выполнения последующего кода.
- **Условный оператор** – конструкция, позволяющая выполнять код только при истинности условия.
- **Цикл** – последовательность повторного выполнения блока кода.
- **`break`** – команда, прерывающая цикл.
- **`continue`** – команда, пропускающая текущую итерацию цикла.
- **`else` у цикла** – блок, исполняемый, если цикл завершился без `break`.
- **Индент** – отступ, определяющий принадлежность кода к блоку.
- **Форматирование условия** – правила расстановки скобок и операторов в условных выражениях.
- **Логический оператор** – `and`, `or`, `not`.
- **Тернарный оператор** – короткая форма условного присваивания.

## Основное содержание

### 1. Условные операторы (30 мин)

#### 1.1. `if` (10 мин)
```python
x = 7
if x > 5:
    print("x больше 5")
```

#### 1.2. `elif` и `else` (10 мин)
```python
score = 85
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
else:
    grade = 'C'
print(grade)
```

#### 1.3. Вложенные условия (10 мин)
```python
age = 20
if age >= 18:
    if age >= 21:
        print("Полноценный взрослый")
    else:
        print("Взрослый, но не совершеннолетний")
```

### 2. Циклические конструкции (30 мин)

#### 2.1. `while` (10 мин)
```python
count = 0
while count < 5:
    print(count)
    count += 1
```

#### 2.2. `for` (10 мин)
```python
for i in range(3):
    print(i)
```

#### 2.3. `break` и `continue` (10 мин)
```python
for i in range(1, 10):
    if i == 5:
        break
    if i % 2 == 0:
        continue
    print(i)
```

### 3. `else` у циклов (10 мин)
```python
for i in range(3):
    print(i)
else:
    print("Цикл завершён без прерывания")
```

### 4. Тернарный оператор и логические выражения (10 мин)
```python
status = "adult" if age >= 18 else "minor"
```

## Примеры и иллюстрации
- **Вычисление факториала**  
  ```python
  def factorial(n):
      result = 1
      for i in range(2, n + 1):
          result *= i
      return result
  ```
- **Проверка числа на простоту**  
  ```python
  def is_prime(n):
      if n <= 1:
          return False
      for i in range(2, int(n**0.5) + 1):
          if n % i == 0:
              return False
      return True
  ```

## Выводы по лекции
Управляющие инструкции – фундаментальный инструмент для контроля потока выполнения программы. Условные операторы позволяют принимать решения, а циклы – повторять действия. Понимание семантики `break`, `continue` и `else` повышает гибкость кода и облегчает отладку. Тернарный оператор и логические выражения дают компактные альтернативы классическим конструкциям.
